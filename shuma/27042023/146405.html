三星Note20系列参数详解

更新时间：2023-04-27 13:01

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

三星Note20系列是2020年三星旗下的旗舰手机产品，包括Note20、Note20 Ultra两款手机。这两款手机在外观设计、硬件配置、摄像头性能等方面都有很大的提升，成为目前市场上最受欢迎的手机之一。接下来，我们来详细了解一下它们的参数。

1.外观设计

三星Note20采用了2.5D玻璃+金属中框的设计，背部采用了全新的Matte AG工艺，让手机更加防滑，易于握持。它的尺寸为161.6 x 75.2 x 8.3 mm，重量为192g。Note20 Ultra则采用了3D曲面玻璃+金属中框的设计，它的尺寸为164.8 x 77.2 x 8.1 mm，重量为208g。

2.屏幕

三星Note20搭载了6.7英寸FHD+ AMOLED屏幕，分辨率为2400 x 1080像素，屏幕刷新率为60Hz，采用了HDR10+技术，色彩表现十分出色。Note20 Ultra则采用了6.9英寸WQHD+ AMOLED屏幕，分辨率为3200 x 1440像素，屏幕刷新率也提升到了120Hz。

3.硬件配置

三星Note20搭载了Exynos 990处理器（或Snapdragon 865+），RAM为8GB，ROM为128GB/256GB，内置4300mAh电池，支持25W有线快充和15W无线快充，还支持无线反向充电。Note20 Ultra则搭载了Exynos 990处理器（或Snapdragon 865+），RAM为12GB，ROM为128GB/256GB/512GB，还内置了4500mAh电池，同样支持25W有线快充和15W无线快充，以及无线反向充电。

4.摄像头

三星Note20搭载了三摄，包括1亿像素主摄、1200万像素超广角摄像头和1200万像素望远摄。前置摄像头为1000万像素。Note20 Ultra则搭载了四摄，包括1亿像素主摄、1200万像素超广角摄像头、1200万像素望远摄和激光对焦模块。前置摄像头为1000万像素。

5.系统与其他功能

三星Note20系列都搭载了最新的One UI 2.5系统，支持5G网络，并且还有WiFi 6、蓝牙5.0、NFC、GPS等功能。它还在S Pen功能上进行了升级，支持向下100ms的延迟，提高了书写的流畅度和响应速度。

6.价格

目前，三星Note20的售价为799美元（约合人民币5600元），Note20 Ultra的售价为1299美元（约合人民币9000元），在高端旗舰手机市场具有很强的竞争力。

总结

三星Note20系列作为三星今年的旗舰手机，拥有了更高的配置和更出色的性能。无论是外观设计还是硬件配置，都有很大的提升。如果你对手机的性能有很高的要求，那么三星Note20系列肯定是你的不二选择。
