三星Galaxy Note i9220详细评测

更新时间：2023-05-14 06:33

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

作为一款有着超大屏幕、独具特色的手机，三星Galaxy Note i9220在发布后备受关注。我们今天来为大家详细评测这款手机。

外观方面，三星Galaxy Note i9220为6.3英寸屏幕，手机尺寸为151.1×80.5×9.4毫米，整体较为修长且颜值不错，而整机重量为178克，较为沉重，不过对于这种屏幕尺寸也算是正常。屏幕方面，i9220采用了三星自家的Super AMOLED HD技术，分辨率为1280×720，色彩呈现惊艳，屏幕下方内置了物理按键，可以流畅使用。

性能方面，三星Galaxy Note i9220采用了双核1.4GHz的处理器，配备1GB运存，内部储存大的版本最高可达32GB，支持microSD扩充，这样的配置可以保证支持多任务、流畅运行各类软件。

系统方面，i9220使用的是基于Android 4.0的TouchWiz 4.0系统，操作起来不会感到生疏，而且三星还针对Galaxy Note这类大屏幕手机进行了优化，增加了多页面的浏览、画廊预览等功能，同时还内置了S Pen手写笔，支持手写、手绘、标记等操作，很实用。

拍照方面，三星Galaxy Note i9220搭载了800万像素的主镜头和200万像素的前置镜头，拍摄效果不算出众，但对于一款大屏幕手机还算有水平。而且三星在软件方面也为其加入了各种拍摄模式和场景，可以满足用户多种拍摄需求。

续航方面，Galaxy Note i9220内置了2500mAh的电池，虽然屏幕较大，但机身尺寸也较为宽厚，所以电池容量与续航表现相当正常，而且还支持快速充电。

总体来说，三星Galaxy Note i9220是一款综合表现不错的大屏幕手机，屏幕、操作、拍照、续航都有表现，而且还有S Pen手写笔的加持，弥补了显示器键盘与鼠标的不足，非常适合文艺、学生党这些群体。
