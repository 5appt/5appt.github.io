苹果公司在2015年9月发布了iPhone 6s系列手机，其中6s和6s Plus是该系列中的两款手机。那么这两款手机的参数、配置和尺寸有哪些差异呢？下面我们来一一探讨。

首先是外观尺寸方面，iPhone 6s的尺寸为138.3 x 67.1 x 7.1 mm（长x宽x厚），重量为143克，而iPhone 6s Plus的尺寸为158.2 x 77.9 x 7.3 mm（长x宽x厚），重量为192克。从外观上看，二者的确存在一定的差异，尤其是尺寸和重量上。

其次是屏幕大小方面，iPhone 6s采用的是4.7英寸Retina HD显示屏，分辨率为1334 x 750像素，而iPhone 6s Plus则采用的是5.5英寸Retina HD显示屏，分辨率为1920 x 1080像素。这也是二者在屏幕大小和分辨率上的体现。

再来看看处理器和操作系统方面的配置，iPhone 6s和6s Plus均采用的是苹果公司自主研发的A9芯片，更加高效和稳定，能够提供更加流畅的使用体验。同时，它们也搭载了最新的iOS 12操作系统，能够支持更多的应用和功能。

接下来是拍照能力方面，iPhone 6s和6s Plus都采用了1200万像素的后置摄像头和500万像素的前置摄像头，能够提供更加出色的拍照和拍摄体验。此外，它们还搭载了很多相机技术，如Live Photos、自动对焦和智能HDR等，为用户提供更加专业的摄影体验。

最后是存储容量方面，iPhone 6s和6s Plus均提供了三种存储容量选择，分别是16GB、64GB和128GB。其中128GB的存储容量能够支持用户对更多的文件和数据进行存储和管理，使手机变得更加实用和便利。

总的来说，iPhone 6s和6s Plus在尺寸、屏幕大小、处理器、操作系统、拍照能力和存储容量等方面都存在一定的差异，用户可以根据自己的需求和实际情况进行选择。无论选择哪一款，都能够为我们带来更加优秀的使用和体验。
