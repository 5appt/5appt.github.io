近年来，随着科技的发展，手机变得越来越普及，品牌也越来越多，而三星作为一家韩国知名的电子产品制造商，早在2013年就推出了许多备受关注的手机产品。那么，当时的三星手机价格究竟如何呢？下面就让我们一起揭秘。

首先，2013年，三星共推出了众多手机产品，包括Galaxy S4、Galaxy Note 3、Galaxy Ace 3等多款产品。其中，Galaxy S4和Galaxy Note 3是当时销售最为火爆的两款产品，价格也是相对较高的。

2013年，Galaxy S4的市场价格为4199元，而Galaxy Note 3的市场价格则为5299元。相比之下，Galaxy Ace 3这样的中低端产品价格则相对较为亲民，大概在1000元到2000元之间。

虽然三星的手机价格相对较高，但它们的性能也是十分强劲的。Galaxy S4搭载了1.9GHz的高通骁龙600处理器和2GB的RAM，拥有5英寸的1080p分辨率屏幕，并且配备了1300万像素的后置摄像头以及2600mAh电池。而Galaxy Note 3则更加高端，拥有5.7英寸的1080p分辨率屏幕、3GB的RAM和3200mAh电池。此外，Galaxy Note 3还具备了S Pen手写笔，可以实现更加精准的操作。

在当时，三星的手机凭借着强大的配置和丰富的功能赢得了广泛的好评。不过，面对如今快速变迁的市场，2013年的三星手机早已成为历史。如今，三星的手机产品已经推陈出新，不断推出更加先进的配置和功能，也让消费者们看到了科技的不断进步和变革。
