金立老款手机型号大全

更新时间：2023-04-28 13:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

金立是一个中国品牌的手机厂商，它的手机以出色的硬件配置和完备的软件环境备受用户欢迎。在其悠久的历史里，金立曾经推出过不少经典机型。下面，让我们来一起回顾一下那些经典的金立老款手机吧！

1.金立 E2

金立 E2 是金立于 2006 年推出的一款基于 3G 技术的移动电话，它采用了时当时最先进的摄像头模块，支持 15 帧的视频录制，并支持最多 2048 级的变焦和自动对焦功能。它还支持 mp3、mp4 多媒体播放。

2.金立 S2

金立 S2 是最早的一款支持 WAP 上网的手机，它采用了时尚、小巧的设计理念。该机的拍照效果也不错，它搭载的 30 万像素 CMOS 摄像头支持自动对焦、连拍等功能。

3.金立 S3

金立 S3 是一款基于 Java 技术的手机，它是一款轻巧的旋转式滑盖机型，支持超大的 2.8 英寸屏幕，搭载了 200 万像素的摄像头，并支持 MP3、MP4 等多媒体播放。

4.金立 DX800

金立 DX800 是一款基于 Windows Mobile 6.1 系统的手机，它是一款拥有双屏幕设计的手机，分别是前后两个屏幕，支持车载 GPS 导航、WIFI、光学识别等功能。

5.金立 G902T

金立 G902T 是金立推出的一款 3G 手机，采用了目前所见最薄的设计理念，只有 8.8 毫米的机身厚度。该机支持 JAVA 应用程序，支持蓝牙无线传输，支持 3D 游戏、MP3、MP4 等音视频播放。

6.金立 W809

金立 W809 是一款搭载了 Android 4.0 系统的智能手机，该机支持了快速充电（5 分钟即可充电 3 个小时），搭载了高通 MSM8625 处理器，并支持 WCDMA/GSM 双卡双待，支持 WIFI、蓝牙、FM 等功能。

7.金立 N900

金立 N900 是一款基于 Android 4.1 系统的智能手机，支持 NFC 短距离通讯技术，采用了 5 英寸高清屏幕，搭载了 1300 万像素的摄像头，支持 WIFI、3G 等功能。

8.金立 S11

金立 S11 是一款基于 Android 4.4.4 系统的智能机，支持全网通，采用了时尚外观设计，搭载了 5.5 英寸大屏幕，支持快速充电、后置 13 像素自动对焦摄像头等诸多功能。

9.金立 M6

金立 M6 是一款搭载了 Android 7.0 系统的智能手机，采用了高通骁龙 435 处理器，支持 4G+、CDLA 等特色功能，支持 16 亿像素的全景拍摄、NFC、多屏互通等新功能。

10.金立 M2017

金立 M2017 是一款搭载了 Android 6.0 系统的极品豪华手机，支持全网通，采用了高通骁龙 820 处理器，搭载了 5.7 英寸双曲面 2K 屏幕，配备了 2x2500 毫安·时的大电池，支持快速闪充等特色功能。

以上就是金立老款手机型号大全的介绍，这些手机在当时都曾经是最具代表性的产品，它们都曾经给消费者带来过不同程度的体验，值得用户回味。
