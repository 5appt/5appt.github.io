Lenovo联想P780：一款实力派手机的全方位评测

更新时间：2023-04-28 07:29

在智能手机市场的激烈竞争下，Lenovo联想P780凭借着出色的性能表现和卓越的续航能力，备受消费者关注。那么这款手机究竟能否真正称得上是一款实力派手机呢？今天，我们将对Lenovo联想P780进行一次全方位的评测，为大家揭开它的面纱。

造型设计：

Lenovo联想P780的整体造型设计简洁大气，采用了2.5D弧面设计，手感舒适，握持稳定。机身正面配备了一块5英寸IPS屏幕，分辨率达到了720P，提供了不错的观感效果。值得一提的是，该屏幕支持多点触控，用户可通过手势操作更加便捷。

在机身背面，Lenovo联想P780采用了磨砂材质，以及一颗位于上方的主摄像头和配套的LED闪光灯。同时，底部还配有一个扬声器，并标有联想的LOGO。整体造型简洁大方，对于追求简单时尚的年轻人来说，或许是个不错的选择。

硬件配置：

Lenovo联想P780配备了一颗MTK6589四核处理器，拥有1GB RAM和4GB ROM的存储空间（可扩展），支持多层次多任务处理。同时，该手机还配备了一颗强大的5000mAh电池，可以确保用户拥有更长的通话和浏览时间。

除此之外，Lenovo联想P780还配备了一颗800万像素的后置摄像头和一颗0.3万像素的前置摄像头，可以满足用户日常的相片和视频聊天需求。此外，该手机还支持USB OTG功能，可以方便地进行U盘扩展存储。

软件体验：

Lenovo联想P780预装了Android 4.2.1操作系统，同时还搭载了联想自家的VIBE UI 1.5定制版。VIBE UI 1.5定制版具有良好的用户体验，支持锁屏快捷方式、自定义主题等多项功能，方便用户进行个性化设置。

此外，该手机配备了联想独有的智能助手--DOit应用，支持备份、恢复、分享等多项功能。为用户提供更加全面的移动助手服务。

综合评价：

从以上的评测来看，Lenovo联想P780确实是一款不错的手机选择。它不仅拥有出色的续航能力和表现优秀的MTK6589处理器，还支持多点触控、USB OTG功能等，足以满足用户对于多任务处理和娱乐需求。同时，VIBE UI 1.5定制版和DOit应用也为用户提供了更加贴心的操作和个性化设置。

总的来说，Lenovo联想P780在性价比方面表现良好，适合那些注重便捷操作和长时间续航的用户选择。唯一的缺点可能就是它的系统更新速度较慢，但是对于多数消费者而言，并不会是非常大的问题。
