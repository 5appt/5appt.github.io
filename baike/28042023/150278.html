一元二次方程的解法求根公式

更新时间：2023-04-28 06:36

大家好，下面小编给大家分享一下。很多人还不知道一元二次方程求根的公式。以下是详细的解释。现在让我们来看看！

首先判断△ = b-4ac，如果△ < 0，则原方程没有实根；一元二次方程的标准型是ax+bx+c = 0，求根公式是x = [-b (b-4ac)]/2a。如果△=0，原方程有两个相同的解，分别是x =-b/2a；如果△ > 0，则为x = (-b+2a)。匹配方法是先将常数c移到方程的右边，然后将二次项系数变为1，再简化为-c/a = (b/2a)。

如果这个方程=0，原方程有两个相同的解，分别是x =-b/2a；如果这个公式大于0，那么x = [-b (b-4ac)在根号下]/2a；直接开平法，形式为(x-m) = n (n > 0)，可直接求出x = m的根号n；因式分解法将标准方程转化为(mx-n)(dx-e)=0的形式，直接得到x=n/m或X = E/D..

只含有一个未知数(一元)且未知数的最高次为2(二次)的积分方程称为一元二次方程。一元二次方程经过排序后可以转化为一般形式AX+BX+C = 0 (A ≠ 0)。

其中AX叫二次项，A是二次项的系数，bx叫线性项，B是线性项的系数，C叫常数项。

一元二次方程有哪些解法？

一元二次方程有四种解法:直接开平法；匹配方法；公式法；因式分解法。解一元二次方程的基本思想方法是化简为两个一元二次方程。

1.直接开平方法

x = p或(NX+m) = p (p ≥ 0)形式的一元二次方程可用直接开平法求解。如果把方程转换成x = p，那么就可以得到x = √ p。如果方程可以转化为(NX+M) = P (P ≥ 0)的形式，那么NX+M = √ P，那么就可以求出方程的根。

2.匹配法:用匹配法求解方程AX+BX+C = 0 (A ≠ 0)。先把常数C移到方程的右边，把二次项系数变成1，在方程的两边加上二次项系数的一半平方，方程的左边就变成了完全平坦的样子。

3.公式法:将一元二次方程转化为一般形式，然后计算判别式△ = b-4ac的值。当B-4ac≥0时，将各种系数A、B、C的值代入公式，求方程的根。

4.因式分解法:将方程一边变形为零，另一边的二次三项式分解为两个线性因子的乘积，使两个线性因子分别等于零，得到两个线性方程。求解这两个线性方程得到的根就是原方程的两个根。

②方程两边除以二次项系数，使二次项系数为1，常数项移至方程右侧；

③方程两边加上第一项系数一半的平方；

④左侧匹配成完全平坦模式，右侧匹配成常数；

(2)匹配法的理论基础是完全平方公式。
