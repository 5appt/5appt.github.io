注册表屏蔽网站

更新时间：2023-04-20 14:04

大家好，下面小编给大家分享一下。很多人还不知道注册表屏蔽网站。下面是详细的解释。现在让我们来看看！

1.在windows7系统中点击“开始”-“运行”，在对话框中输入“regedit”进入注册表编辑器。

2.IE主页的注册表文件放在:HKEY _当前_用户\软件\微软\ Internet Explorer \ main \起始页下，该子项的键值是IE主页的网址。关键首页可以修改，可以修改，用户可以改成自己常用的网站或者“关于:空白”，也就是空白页。这样重启IE就能看到效果；

3.如果这种方法不管用，那是因为某些病毒或流氓软件在你的电脑里安装了一个自运行程序。即使通过修改注册表来恢复IE主页，只要一重启电脑，程序就会自动运行并再次篡改。

4.这时我们需要对注册表文件进行更多的修改，运行“regedit”，然后依次展开HKEY _本地_机器\软件\微软\ Windows \当前版本\运行主键，然后删除其下未知的promoter键值，再相应删除其未知的自运行程序文件，最后从IE选项中重置起始页。

5，但是如果重启，你会发现打开ie又变回了2345；这是因为有些IE的主页被改了，即使设置了“使用默认页面”，仍然无效。这是因为IE的起始页的默认页面也被篡改了。在这种情况下，我们也可以通过修改注册表来解决。

6.运行“regedit”展开:HKEY _ local _ machine \ software \ Microsoft \ Internet Explorer \ main \ Default_Page_URL子键，然后只需更改“Default _ Page _ UR”子键的键值中那些被篡改的网站，或者设置为IE的默认值即可；

7.还有一点就是看系统启动项，进入msconfig看启动项。在这里，您可以找到每次启动时访问的网站的URL地址。还有一点就是在启动栏里有一个到这台机器的连接，可能是连接了一个*。js文件。找的时候要小心。删除左边的复选框后，删除*。JS文件，然后重启系统。

以上解释了注册表屏蔽网站。这篇文章已经分享到这里了，希望对大家有所帮助。如果信息有误，请联系小编进行更正。
