win10任务栏全屏不消失的解决方法

日期：2023-04-17 栏目：百科知识 浏览：次

大家好，下面小编给大家分享一下。很多人还不知道win10任务栏不会全屏消失的解决方案。以下是详细的解释。现在让我们来看看！

当Windows 10任务栏不工作时，它可能以下列方式之一显示:

单击任务栏无效。 图标不显示。 键盘快捷键不起作用。

注意:本文中的说明仅适用于Windows 10。

Windows 10任务栏不响应的原因

Windows 10任务栏可能因为各种原因被冻结，包括操作系统更新不完整、更新错误、系统文件损坏或用户帐户文件损坏。

如何让Windows 10冻结任务栏工作

01 使用任务管理器重新启动Windows资源管理，在键盘上执行此操作:

1按Ctrl+Shift+Esc打开任务管理器。

2如有必要，单击更多详细信息查看所有选项卡。

3选择进程选项卡，然后单击Windows资源管理器。

4选择Windows资源管理器后，单击重新启动窗口的底部。

02 运行SFC扫描。

1在键盘上，按Ctrl+Shift+Esc打开任务管理器。

2选择文件 > “运行新任务”。

3在创建新任务对话框中，键入资源管理器，然后单击确定。

4在文件资源管理器中，导航到 c: Windows System32 。

5找到cmd.exe并右键单击它。

6从菜单中选择以管理员身份运行 > 是。

7在下一个窗口中，在提示符下键入sfc/scannow，并等待100%验证。

03使用Powershell。这是一个由两部分组成的过程。首先，启用Windows防火墙。

1按键盘上的Ctrl+Shift+Esc。

2如有必要，单击更多详细信息查看

3选择服务选项卡，然后在名称下，查找 MPSVC (Windows防火墙)。

4确保图标“打开服务”显示在其旁边。如果没有，右键单击它并选择 Go 。

现在，按键盘上的Win+R(Win是Windows徽标键)。在运行对话框中，键入powershell并按Enter。在提示符下，键入以下命令，然后按Enter :

Get-appx package-all users | Foreach { Add-appx package-disabled development mode-Register " $($ _ .install location)app xmanifest . XML " }

04运行DISM映像管理命令。

1在键盘上，按Ctrl+Shift+Esc打开任务管理器。

2选择文件 > “运行新任务”。

3在创建新任务对话框中，键入explorer，然后单击确定。

4在文件资源管理器中，导航到 c: Windows System32 。

5找到cmd.exe并右键单击它。

6从菜单中选择以管理员身份运行 > 是。

7在下一个窗口中，在提示符下键入DISM/在线/清理-图像/恢复健康，然后按回车键。等到验证100%。

05启用用户管理器。

1按键盘上的win+r(win是Windows徽标键)。

2在运行对话框中，键入services.msc，然后按Enter 。

3在名称 下，找到并双击用户管理器。

4从启动类型菜单中，选择自动并单击确定。

5.重新启动计算机。

06禁用最近打开的项目。

1按键盘上的Win+I(Win是Windows徽标键)打开设置。

2选择个性化。

3在左栏中，选择开始。

4当打开时，最近打开的项目显示在跳转列表中，或者任务栏上的开关关闭。

07运行系统还原。

1首先，按Win+R(Win是Windows徽标键)。

2在运行对话框中，键入sysdm.cpl，然后按Enter 。

3在系统属性对话框中，选择系统保护 > 系统还原。

4在下一个屏幕上，选择下一步，然后选择系统还原点。

5按照屏幕上的说明继续，然后重新启动系统。

08 创建新的用户帐户。

1按Win+I(Win是Windows徽标键)打开设置。

2选择账户 > 家庭和其他。

3然后选择将其他人添加到这台电脑。

4选择我没有这个人的登录信息，然后选择下一步。

5选择添加没有Microsoft帐户的用户，然后选择下一步。

6输入帐户的用户名和密码，然后选择下一步。

登录到新的用户帐户。如果任务栏有效，将文件转移到新帐户并删除旧帐户。
