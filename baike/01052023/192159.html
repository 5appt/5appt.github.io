台式电脑忘记密码怎么解决视频

更新时间：2023-05-01 18:17

大家好，下面小编给大家分享一下。台式电脑忘记密码，很多人还不知道怎么解决视频。以下是详细的解释。现在让我们来看看！

1.启动电脑，开机画面出现后立即按F8，选择【带命令行的安全模式】；

2.运行过程结束时，系统列出系统超级用户【管理员】和本地用户【您之前的用户名】的选择菜单，点击【管理员】进入命令行模式。

3.键入命令:[net user your previous user name 123456/add]将[your previous user name]的密码更改为[123456]。如果要在这里添加新用户(比如用户名为abcdef，密码为123456)，请键入【网络用户abcdef 123456 /add】，添加后可以使用【网络本地组管理员ABCDEF/Add】命令将用户提升为拥有超级权限的系统管理组【管理员】；

4.重启电脑，选择正常模式运行，可以用修改后的密码【123456】登录之前的用户名；

5.或者删除DOS下的SAM文件(这个文件是XP登录用户的密码，在c:\windows\repair下)然后重启。可以使用空密码进入。
