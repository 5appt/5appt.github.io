电脑报runtime error

日期：2023-04-18 栏目：百科知识 浏览：次

大家好，下面小编给大家分享一下。很多人还不知道电脑报告运行时出错。以下是详细的解释。现在让我们来看看！

打开桌面左下角的开始菜单，在搜索框中输入regedit，按回车键打开注册表。同时按[ctrl+f]弹出搜索框，进入运行时搜索，然后右键单击查询到的运行时记录信息，选择[删除]全部删除，也可以在HKEY _ local _ machine \ software \ Microsoft \ Windows \ current version \ Run中删除，删除相当于运行时记录的信息摘要即可解决。

如何解决电脑中的runtimeerror？

1.打开电脑左下角，进入regedit，打开注册表。

2.同时按CTRL+F查询快捷键，会出现一个搜索框。输入要搜索的运行时间。

3.右键单击查询出的运行时记录信息，选择“删除”，逐个删除。

4.如果运行时的记录信息特别大，可以通过删除HKEY _本地_机器\软件\微软\ Windows \ currentversion \ run位置的运行时信息汇总来解决。

以上是如何解决电脑上runtimeerror的介绍，希望对小伙伴们有所帮助。
