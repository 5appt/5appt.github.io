平菇怎么做好吃？教你几招轻松入手

更新时间：2023-05-07 06:26

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

平菇是很多人喜欢的食材，营养价值高，味道鲜美，而且烹饪方法多样。如果你还不知道平菇怎么做好吃，那就不妨试试以下的几招吧。

一、烤平菇

材料：平菇、大蒜、盐、黑胡椒、橄榄油、柠檬汁。

步骤：

1. 清洗平菇，将蒂部切掉。

2. 把大蒜切成小块，放入烤盘中。

3. 把已经清洗好的平菇与大蒜放入烤盘，加入少量盐和黑胡椒。

4. 用橄榄油涂抹在平菇上。

5. 将烤盘放入烤箱中，温度为180度，烤15分钟左右。

6. 烤好之后，滴上柠檬汁即可食用。

二、炒平菇

材料：平菇、姜、葱、蒜、盐、鸡精、生抽、油。

步骤：

1. 清洗平菇，切成片。

2. 姜、葱、蒜分别切成末。

3. 热锅凉油，加入姜末和蒜末煸炒出香味。

4. 加入平菇，翻炒。

5. 加盐、鸡精、生抽调味即可。

三、煲汤

材料：平菇、鸡骨、红枣，枸杞。

步骤：

1. 鸡骨放入水中煮，煮至汤色变清，鸡骨去掉后备用。

2. 平菇清洗干净，红枣和枸杞洗净。

3. 把鸡汤倒入锅中，加入平菇、红枣、枸杞，煮开后转为小火煲1小时。

4. 煮好后加盐即可。

四、拌菜

材料：平菇、西红柿、青椒、盐、生抽、香油。

步骤：

1. 平菇清洗干净，西红柿、青椒洗干净后切丁备用。

2. 将切好的平菇放入开水中焯水，捞出沥干，备用。

3. 热锅凉油，将切好的西红柿和青椒下锅翻炒，加盐和生抽调味。

4. 将炒好的青红椒拌入焯好水的平菇中，撒上些许香油即可。

总结：

平菇是一种非常健康的食材，不仅味道鲜美，而且功能特别丰富。采用不同的烹饪方法可以制作出不同口味的平菇。无论是烤平菇、炒平菇、煲汤还是拌菜，只要你跟着上述步骤操作，幸福的味蕾就会被平菇征服！
