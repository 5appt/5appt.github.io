鲁菜菜谱大全——品味山东美食的精华

更新时间：2023-04-10 06:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

山东是中国四大菜系之一，其中鲁菜以其深厚的文化底蕴、丰富的品类和独特的风味成为了国内外人们喜爱的菜系之一。鲁菜以其口感道地、味道清爽、入口鲜美、回味无穷而闻名于世。在山东，每个人都会烹饪一些家常鲁菜，这也是鲁菜家常化、普及化的重要原因之一。

下面是鲁菜菜谱大全，您可以根据个人口味或者材料喜好选择不同的鲁菜，享受纯正的山东美食。

一、主食类

1. 烙饼

【原料】面粉500克，温水300毫升，油适量。

【做法】

1. 将面粉倒入盆中，慢慢加入温水揉成面团，揉到表面光滑。

2. 将揉好的面团分割成20个小面团。

3. 将小面团擀成薄饼，注意饼面不能擀破。

4. 在平底锅中加油，烧热后将饼放入锅中，小火煎至两面金黄即可。

5. 完成后食用即可。

2. 杂粮小米粥

【原料】小米200克，水适量，大米100克，白糖适量。

【做法】

1. 将小米淘洗干净，加水煮开后转小火，煮至小米烂熟。

2. 将大米洗净，与小米混合后加适量水，大火煮开后改小火煮至粥成。

3. 最后加白糖调味即可。

二、肉类菜肴

1. 鲁菜三味

【原料】五花肉150克，熟猪皮50克，豆腐干50克，青辣椒适量，红辣椒适量，姜片适量，大葱适量，蒜头适量，花椒适量，盐适量，生抽适量，料酒适量。

【做法】

1. 将五花肉切成大小均匀的薄片，姜片和葱切末，蒜头拍碎。

2. 熟猪皮切成片状，豆腐干切成小块状。

3. 葱末和姜末在碗中调和成姜葱汁。

4. 锅中烧油，加入花椒和姜蒜炒香，加入薄片肉煸炒至变色。

5. 加入豆腐干、青红辣椒段，用大火翻炒，加入姜葱汁、料酒、生抽和盐继续翻炒均匀。

6. 最后加入熟猪皮块翻炒，起锅装盘即可。

2. 葱爆牛肉

【原料】牛后腿肉150克，葱4根，姜10克，蒜5克，花椒适量，盐适量，生抽适量，料酒适量。

【做法】

1. 将牛腿肉切成小块，在碗中加入料酒、盐、生抽腌制10分钟。

2. 葱切成葱花，姜切小丝，蒜头拍碎备用。

3. 在锅中加入油，烧热后加入花椒炒香，将牛肉块放入锅中爆炒至变色后，加入姜、蒜、葱花炒香。

4. 将撒上盐和少量料酒，翻炒均匀后即可。

三、海鲜类菜肴

1. 清蒸大闸蟹

【原料】大闸蟹8只，大葱3根，姜2小块，油适量，盐适量，料酒适量。

【做法】

1. 将大葱切成段状，姜切成片状。

2. 将大闸蟹彻底清洗干净，将前爪解掉，将大葱丝和姜片塞入蟹壳内。

3. 在大蒸锅中加水，加入适量盐和料酒，大火蒸10分钟。

4. 取出大闸蟹，前爪放回原位，撒上葱末即可。

2. 爆炒虾仁

【原料】大虾仁300克，青椒适量，红椒适量，生姜适量，葱适量，盐适量，白胡椒粉适量，生抽适量。

【做法】

1. 所有青红椒、生姜切丝，葱切末。

2. 虾仁去壳待用。

3. 热锅加油，倒入青红椒、姜丝翻炒，加入虾仁翻炒至变色。

4. 加入葱末、盐、白胡椒粉、生抽翻炒均匀即可。

四、蔬菜类菜肴

1. 地三鲜

【原料】土豆3个，茄子2个，辣椒1个，蒜头适量，姜适量，盐适量，生抽适量，水淀粉适量。

【做法】

1. 土豆去皮，切块，茄子切片状，辣椒切段状。

2. 热锅加油，先放土豆煸炒，再放茄子煸炒至变色。

3. 加入适量盐、生抽翻炒均匀。

4. 在锅中再烧油，加入蒜头、姜片、辣椒煸炒，倒入水淀粉，加入炒好的菜再翻炒均匀即可。

2. 炒莲白

【原料】莲白300克，生姜适量，蒜头适量，盐适量，生抽适量。

【做法】

1. 莲白切成片状，姜切成丝，蒜拍碎待用。

2. 热锅加油，加入姜片蒜末煸炒出香味。

3. 将莲白放入锅中炒，炒至断生时，倒入生抽。

4. 加入适量盐，翻炒均匀即可。

五、汤类菜肴

1. 八宝鸭

【原料】鸭1只，糯米适量，莲子适量，百合适量，红枣适量，桂圆适量，干贝30克，冬菇5朵，盐适量，生姜适量。

【做法】

1. 将鸭子宰杀，掏干净，浸泡在盐水中刷洗干净，切块。

2. 再将所有配料洗净，糯米淘净备用。

3. 八宝用开水焯水后沥干水分，鸭子入油锅煎脆。

4. 煎的鸭子块放在锅底的气锅中。

5. 把八宝混合好，放进整只鸭子里，加清水盖过所有食材，开大火。

6. 盖上气锅盖煮开，再开小火煮20分钟即可。

2. 豆腐鲫鱼汤

【原料】鲫鱼1只，豆腐1块，葱适量，姜适量，盐适量，香油适量，料酒适量。

【做法】

1. 鲫鱼去鳞，洗净，拍干水分，切成约3厘米的鱼块。葱切段，姜切片。

2. 将豆腐切成厚片状，用热水烫过后捞出备用。

3. 煮开一锅水，在沸水中加入葱姜片，鲫鱼块和料酒，闷煮5分钟。

4. 将豆腐块加入汤中，继续煮1分钟后加入盐调味。

5. 最后淋上香油即可。
