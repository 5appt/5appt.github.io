猪肘子的做法大全

更新时间：2023-04-14 06:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

猪肘子是一道可以滋养身体的美味佳肴，营养价值非常高，特别适合在寒冷的冬季食用。烹制猪肘子的方式有很多种，以下将介绍几种简单易做的猪肘子做法。

1. 红烧猪肘子

材料：猪肘子、姜片、葱段、料酒、酱油、糖、豆腐、香菇、胡萝卜、冰糖、桂皮、八角。

做法：

1. 将猪肘子切成块，放入开水中汆烫3分钟。

2. 把猪肘子取出，洗净血沫，沥干备用。

3. 香菇、胡萝卜切片，豆腐切块。

4. 干热锅冷油，倒入冰糖，小火炒至冰糖溶化，并呈金黄色时加入姜片、葱段、桂皮、八角，再翻炒几下，加入猪肘子煸炒至表面金黄。

5. 倒入料酒、酱油、糖，加入水没过猪肘子，加盖大火烧开后转小火炖煮30分钟。

6. 加入香菇、胡萝卜、豆腐，炖煮10分钟。

7. 最后加入少许盐调味即可。

2. 炭烤猪肘子

材料：猪肘子、盐、黑胡椒、蒜末、鸡精。

做法：

1. 提前将猪肘子在内侧切开，去除里面的骨头，切成约2厘米厚的片状。

2. 将猪肘子片用盐、黑胡椒、蒜末、鸡精腌制10分钟。

3. 将猪肘子片平铺在炭火上，烤制5分钟后翻面再烤5分钟（时间根据火力大小而定）。

4. 烤熟后将猪肘子片取出，切成小块即可食用。

3. 清炖猪肘子

材料：猪肘子、姜片、葱段、料酒、盐、胡萝卜、香菇。

做法：

1. 将猪肘子切成小块，汆烫至表面颜色变白，用清水冲洗干净，去除血沫。

2. 姜切片，葱切段，胡萝卜、香菇洗净备用。

3. 热锅加油，烧热后放姜片和葱段爆香，放入猪肘子煸炒至微黄。

4. 倒入料酒，切入胡萝卜、香菇，加适量水，越过猪肘子。

5. 然后加盐适量，盖上锅盖煮30分钟左右即可。

4. 红烧猪蹄爪

材料：猪蹄爪、姜片、葱段、料酒、酱油、老抽、生抽、糖。

做法：

1. 猪蹄爪洗净洗干净后，用开水焯水。

2. 焯水后捞出，用清水冲洗干净。

3. 锅中倒入适量的油，待油热后下入姜片和葱段炒香。

4. 再次倒入猪蹄爪煸炒均匀，至表面微黄时加入料酒。

5. 加入酱油、老抽和生抽，略加翻炒，之后放入糖。

6. 加水没过猪蹄爪，大火烧开后转小火慢炖45分钟左右，最后加盐调味即可。

总结：

猪肘子是一道应季美食，选用好的材料，搭配适当方法，可以制作出多种口味美味的猪肘子料理。以上介绍的几种做法，简单易学，营养丰富，尤其适合冬季食用，有助于保持身体健康。
