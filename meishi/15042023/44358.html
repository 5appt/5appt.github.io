如何制作美味可口的凉菜——经典凉菜的食材和做法大全

日期：2023-04-15 栏目：美食常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

凉菜是夏季的常见菜品，清爽而又美味。它不仅能够解暑降温，而且可以增加胃口，开胃消食，受到广大食客的喜爱。今天，我们就一起来学习一些经典的凉菜的食材和做法吧！

一、韭菜花拌豆腐

食材：

豆腐200克

韭菜花50克

盐适量

鸡精适量

香菜适量

做法：

1. 豆腐切成小块，用开水焯烫2分钟。

2. 把豆腐盛出，放入盆中，加入适量盐和鸡精拌匀，静置5分钟。

3. 韭菜花焯水后捞出，切成小段。

4. 将豆腐和韭菜花拌匀，加入香菜末即可。

二、凉拌茄子

食材：

茄子2个

小葱适量

姜适量

蒜适量

辣椒适量

醋适量

酱油适量

做法：

1. 茄子洗净切块，放入沸水中煮熟。

2. 捞出茄子沥干水分，放入盘中。

3. 将小葱、姜、蒜、辣椒切碎，放入碗中。

4. 加入醋、酱油，搅拌均匀，然后淋在茄子上即可。

三、拌凉粉

食材：

凉粉300克

大葱适量

辣椒适量

姜适量

醋适量

酱油适量

糖适量

盐适量

做法：

1. 凉粉切成丝，放入开水中焯水后沥干。

2. 大葱切成花状，姜切成丝，辣椒切成小段。

3. 在碗中加入醋、酱油、糖、盐、姜丝、辣椒段，搅拌均匀。

4. 将凉粉和拌好的调料一起拌匀，放入冰箱冷藏1小时后即可食用。

四、凉拌海带丝

食材：

海带丝200克

葱姜蒜适量

盐适量

醋适量

糖适量

味精适量

香菜适量

做法：

1. 将海带丝入沸水中焯烫4分钟左右，放入冷水中过凉，沥干水分。

2. 将葱姜蒜剁碎，加入盐、醋、糖、味精中拌匀。

3. 将拌好的调料倒入海带丝中拌匀，加入香菜末即可。

五、凉拌黄瓜

食材：

黄瓜200克

葱姜蒜适量

盐适量

醋适量

味精适量

辣椒适量

香菜适量

做法：

1. 将黄瓜洗净后，切成细条，加入盐拌匀，静置片刻。

2. 将葱姜蒜剁碎，加入醋、味精、辣椒拌匀。

3. 将黄瓜挤干出水，加入拌好的调料中拌匀，加入香菜末即可。

六、凉拌莴笋

食材：

莴笋200克

葱姜蒜适量

盐适量

醋适量

味精适量

辣椒适量

花生米适量

做法：

1. 莴笋洗净后，切成丝，加入盐拌匀，静置片刻。

2. 将葱姜蒜切碎，加入醋、味精、辣椒拌匀。

3. 把莴笋丝挤干水分，加入拌好的调料中拌匀，加入花生米即可。

以上就是几款经典的凉菜，制作简单，美味可口，清爽可口，十分适合夏季消暑用。大家不妨试试，快来一起来尝试一下吧！
