炖鱼的家常做法

更新时间：2023-04-13 06:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

炖鱼是许多家庭非常喜爱的一道菜，既美味又营养。在制作炖鱼时，要选用新鲜的鱼肉，加上适当的调料和配菜，让它更加美味可口。下面就来介绍几种简单易学的炖鱼的家常做法。

1. 红烧鲫鱼

材料:

鲫鱼1条（约400克），生姜，蒜，葱，香叶，八角，料酒，酱油，白糖，盐。

步骤:

1. 鲫鱼去内脏，清洗干净。

2. 锅内放油，油热后加入生姜、蒜、葱、香叶、八角煸炒出香味，再加入料酒、酱油、白糖、盐和适量的水，烧开后放入鲫鱼，加盖小火炖煮20分钟左右。

3. 调味后收汁，翻身让两面都裹上汁水即可。

2. 清蒸鲈鱼

材料:

鲈鱼1条（约500克），生姜，葱，盐，料酒，味精，香油，花生油。

步骤:

1. 鲈鱼去内脏，清洗干净，用盐、料酒、味精腌制15分钟。

2. 把葱切成丝，姜切成丝。

3. 把鲈鱼放在蒸锅中，水开后蒸10分钟左右，用筷子插入鱼肉，看鱼肉的变色程度，熟度自己掌握。

4. 熟鱼出锅，撒上葱姜丝，白胡椒粉和香油，用热油淋上，即可食用。

3. 酸菜鱼

材料:

鸭血，酸菜，辣椒，姜，蒜，葱，青椒，鲈鱼片，白胡椒粉，盐，食用油。

步骤:

1. 鱼片放盐、白胡椒粉，抓匀腌制10分钟。

2. 酸菜洗净切成小段，青椒、辣椒、姜、蒜都切成小段，备用。

3. 热锅倒油，油热后加入酸菜煸炒，加入适量的水烧开后放入姜蒜葱炒香，再放入青椒、辣椒等翻炒均匀。

4. 放入鱼片煮熟后放入鸭血继续煮2分钟即可。

4. 砂锅鱼头豆腐汤

材料:

鱼头，豆腐，海鲜菇，扁豆，小青菜，生姜，蒜，料酒，酱油，盐，胡椒粉，食用油。

步骤:

1. 鱼头加盐、料酒、姜腌制10分钟。

2. 扁豆去边切段。

3. 锅内倒入适量油，油热后加葱姜蒜爆香，放入鱼头煎至两面金黄色。

4. 加入料酒、酱油、盐和水，烧开后加入豆腐、海鲜菇、扁豆，大火煮至汤汁浓稠即可。

5. 上桌前放入青菜，撒上胡椒粉，热气腾腾的砂锅鱼头豆腐汤就做好了。

5. 清炖草鱼汤

材料:

草鱼1条（约500克），生姜，蒜，红枣，枸杞，淮山片，当归片，盐。

步骤:

1. 草鱼去内脏、鳞，清洗干净。

2. 把所有材料切块备用。

3. 锅内加入适量水、姜、蒜、红枣、枸杞、淮山片、当归片，烧开后放入草鱼，大火煮沸后转小火煮30分钟左右。

4. 炖好后把鱼捞出，取出草鱼肉切成片，趁热撒盐，热汤中再放回去，熄火焖20分钟即可。

总之，制作炖鱼很简单，只要掌握好食材搭配、火候和调料的比例，就能制作出口感鲜美、味道鲜美的炖鱼佳肴。相信在家里试着做一次不同的炖鱼，会给大家带来愉悦的感觉。
