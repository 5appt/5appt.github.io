山药的做法大全

更新时间：2023-04-12 12:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

山药又名淮山，是一种常见的食材，具有丰富的营养和药用价值。它被誉为“补虚佳品”，可以滋补肝肾、强筋健骨、补血养颜等。下面为大家介绍几种山药的做法大全，让您尽情享受它的美味。

一、山药红枣糯米粥

材料：山药100克、红枣20克、糯米100克、冰糖适量

做法：

1. 将糯米淘净后浸泡1小时左右。

2. 山药去皮切小块状，红枣去核洗净备用。

3. 将泡好的糯米加入适量的清水，开大火煮沸后转小火熬煮30分钟，期间要经常搅拌，防止粘锅。

4. 将山药和红枣加入锅内，再煮10分钟左右。

5. 最后加入适量冰糖，再煮5分钟左右即可。

二、山药炖羊肉

材料：山药200克、羊肉适量、姜片适量、葱段适量、料酒适量、盐适量

做法：

1. 羊肉洗净切块，放入沸水中焯水，捞出洗净。

2. 山药去皮切块，姜、葱切片备用。

3. 取一锅加入适量水，放入焯好水的羊肉块，加入姜片、葱段和料酒，开大火烧开后撇去浮沫。

4. 将山药加入锅中，再次烧开后转小火，炖煮1小时左右，期间加盐调味即可。

三、山药莲子糖水

材料：山药100克、莲子50克、红枣适量、冰糖适量

做法：

1. 将莲子泡发后去皮，山药去皮切小块状，红枣洗净备用。

2. 锅中加水，将冰糖放入锅中烧开，放入泡发好的莲子和红枣，煮5分钟左右。

3. 最后加入山药继续煮5分钟左右，等山药变软后即可关火食用。

四、山药蘑菇炒肉片

材料：山药100克、鲜鸡肉适量、蘑菇适量、葱姜适量、盐适量、糖适量、料酒适量

做法：

1. 鸡肉洗净切片，用盐、糖、料酒腌渍15分钟。

2. 山药去皮切片，蘑菇洗净切片，葱姜切末备用。

3. 坐锅热油，放入鸡肉片爆炒至变色，加入蘑菇翻炒，再加入山药翻炒至变色。

4. 加入适量的盐，继续翻炒均匀即可。

五、山药炖老鸭汤

材料：山药100克、老鸭1只、姜片适量、葱段适量、料酒适量、盐适量

做法：

1. 老鸭去内脏后洗净切块，用沸水焯水后捞出备用。

2. 山药去皮切块，姜、葱切片备用。

3. 热锅加油，将鸭块煸炒至表皮金黄，加入姜和葱翻炒几下。

4. 加入料酒、盐和适量的水，大火烧开后转小火焖煮1小时。

5. 最后加入山药煮5分钟左右即可。

六、山药芋头鸭脯汤

材料：山药100克、芋头100克、鸭脯适量、姜片适量、葱段适量、料酒适量、盐适量

做法：

1. 鸭脯一洗净切块，用沸水焯水后捞出备用。

2. 山药去皮切块，芋头去皮切块，姜、葱切片备用。

3. 热锅加油，将鸭块煸炒至表皮金黄，加入姜和葱翻炒几下。

4. 加入料酒、盐和适量的水，大火烧开后转小火焖煮1小时。

5. 最后加入山药和芋头，继续焖煮30分钟即可。

七、山药蒜苔炒肉

材料：山药100克、肉片适量、蒜苔适量、葱姜适量、盐适量、糖适量、料酒适量

做法：

1. 肉片用盐、糖、料酒腌渍10分钟。

2. 山药去皮切块，蒜苔洗净切段，葱姜切末备用。

3. 坐锅热油，放入肉片爆炒至变色，加入蒜苔翻炒，再加入山药翻炒至变色。

4. 最后加入适量的盐，继续翻炒均匀即可。
