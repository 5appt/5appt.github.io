白菜的做法

日期：2023-04-09 栏目：美食常识 浏览：次

白菜是我们日常餐桌上不可或缺的食材，其营养丰富，味道鲜美，可以炒、煮、炖、拌、腌等多种做法。下面就为大家介绍几种口感独特、简单易学的白菜美食做法。

一、白菜拌豆腐干

材料：白菜、豆腐干、盐、鸡精、生抽、蒜蓉、姜丝、香菜

做法：

1. 将白菜修剪干净后切成丝状，洗净沥干水分备用。

2. 豆腐干切块，姜丝和蒜蓉切末。

3. 将白菜和豆腐干放在一个碗中，加入盐、鸡精、生抽、姜丝和蒜蓉，拌匀。

4. 放置20分钟让白菜和豆腐干充分吸收调味汁。

5. 最后撒上香菜即可。

二、红烧白菜

材料：白菜、猪肉、豆瓣酱、姜、大葱、料酒、生抽、盐、糖、鸡精、蒜蓉

做法：

1. 猪肉切成小块，放入开水中焯水备用，白菜洗净切成大块，大葱姜切好备用。

2. 坐锅加油，加入糖炒至融化，然后加入姜葱爆香，接着加入豆瓣酱和蒜蓉，炒出红油后加入猪肉片炒匀。

3. 加适量料酒和生抽炒匀，加入开水或高汤，放入白菜块，加入盐、糖、鸡精，焖煮5分钟即可。

三、蒜蓉白菜炒虾仁

材料：白菜、虾仁、蒜蓉、盐、味精、生抽、糖、淀粉、料酒、鲜汤

做法：

1. 把白菜洗净切成丝，泡在清水中20分钟，然后沥干水分备用。

2. 将虾仁洗净去壳，加入盐、料酒拌匀腌制10分钟。

3. 热锅冷油，加入虾仁煸炒至变色捞出备用。

4. 锅中留少许底油，加入蒜蓉爆香，加入白菜翻炒3分钟，加入适量的鲜汤焖3分钟。

5. 最后加入虾仁、盐、味精、生抽和少许糖，翻匀收汁。

四、酸菜白菜豆腐汤

材料：酸菜、白菜、豆腐、猪肉、辣椒、姜、葱、鸡精、盐、白胡椒、料酒

做法：

1. 将猪肉切成小块，做成五花肉或者肉末备用，白菜、酸菜、豆腐洗净备用。

2. 锅中倒入适量水，放入姜片、葱段，放入猪肉小火煮10分钟。

3. 把酸菜洗净切成小段，放入锅中，煮5分钟，再放入白菜、豆腐块一起煮。

4. 放入盐、鸡精、白胡椒粉、料酒慢炖20分钟。

5. 涨锅前5分钟加入辣椒面搅拌均匀，汤汁收浓后关火即可。

五、糖醋白菜

材料：白菜、白糖、白醋、油、盐

做法：

1. 把白菜洗净，把头抠开。

2. 油热后放入白菜，小火炸至金黄色捞出沥干油。

3. 另起锅，加入白糖和白醋开火烧至糖溶解。

4. 把炸好的白菜放入糖醋水中，翻动几下即可。

5. 最后加入适量盐，翻匀即可上桌。

六、泡椒白菜炒肉片

材料：白菜、猪肉片、泡椒、姜蒜、料酒、生抽、鸡精

做法：

1. 将白菜洗净切成长条状备用，猪肉切片加入料酒、生抽、鸡精腌制15分钟。

2. 锅热加油，下入腌制好的猪肉片快速翻炒，颜色变深后捞出备用。

3. 锅中留少许底油，加入姜蒜和泡椒翻炒出香味，加入白菜翻炒1-2分钟。

4. 最后加入炒好的猪肉块翻炒至熟，根据个人口味加入适量盐、味精即可。

以上几道白菜美食做法，操作简单，味道都特别好，是各位厨艺小白必备的美食大全，不妨一试!
