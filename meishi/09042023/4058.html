鸡胗做法大全：从清炒到炸至香嫩的八种口感

更新时间：2023-04-09 19:42
[db:TAG标签][0]

鸡胗，又称鸡肫，是指鸡的胃部分，其肉质细嫩，口感鲜美。在我国南方地区，鸡胗是一道常见的食材，烹调方式也五花八门。今天，小编就为大家介绍八种不同口感的鸡胗做法。

一、清炒鸡胗

食材：鸡胗一份（约400克）、小葱一根、姜蒜适量、盐适量、料酒适量、生抽适量、白胡椒粉适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，用厨房纸将多余的水分擦干；

2. 将鸡胗切成薄片或条状备用；

3. 热锅凉油，放入葱姜蒜爆香；

4. 将鸡胗倒入锅中，煸炒至变色；

5. 加入盐、料酒、白胡椒粉、少许生抽调味，翻炒均匀即可。

清炒鸡胗，清爽不腻，适合作为凉拌菜或配菜食用。

二、红烧鸡胗

食材：鸡胗一份（约400克）、生姜适量、葱适量、料酒适量、老抽适量、生抽适量、白糖适量、盐适量、八角适量、桂皮适量、干辣椒适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，切成厚片；

2. 生姜切片，葱洗净切段备用；

3. 热锅凉油，放入干辣椒、八角、桂皮爆香；

4. 放入鸡胗煸炒至变色，加入料酒、老抽、生抽、白糖、盐翻炒均匀；

5. 加入适量清水，放入生姜、葱段，小火慢炖至鸡胗入味软烂即可。

红烧鸡胗汁浓味鲜，配米饭最为下饭。

三、蒸鸡胗

食材：鸡胗一份（约400克）、葱适量、姜适量、盐适量。

做法：

1. 将鸡胗用清水冲洗干净，切成小片；

2. 生姜切片，葱洗净切成葱花；

3. 将鸡胗倒入碗中，加入盐、生姜片、葱花拌匀，腌制15分钟；

4. 取锅，倒入适量清水，放入蒸架，开火将水烧开；

5. 将鸡胗放入蒸盘里，放入蒸架中，盖上锅盖，大火蒸10分钟即可。

蒸鸡胗不需添加油腻，清爽健康。

四、炸鸡胗

食材：鸡胗一份（约400克）、盐适量、黄油适量、面粉适量、鸡蛋适量、面包屑适量、油适量。

做法：

1. 将鸡胗用清水冲洗干净，斜切成厚片备用；

2. 按照平常的油炸流程，将面包屑、面粉分别放入两个碗中，另外准备一个打散的鸡蛋碗，再准备一个铺有面粉碗的盘子备用；

3. 将鸡胗片逐个沾上面粉，然后蘸上鸡蛋液，在面包屑碗中翻滚均匀，将面包屑均匀沾附在鸡胗片上；

4. 热锅凉油，放入鸡胗炸至金黄色；

5. 取另一锅放入黄油，稍微加热一下直到融化，将炸好的鸡胗放入黄油锅中翻炒均匀。

炸鸡胗外酥里嫩，香气扑鼻。

五、鸡胗炒豆芽

食材：鸡胗一份（约400克）、豆芽适量、清水适量、姜末适量、葱花适量、盐适量、鸡精适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，切成小块备用；

2. 豆芽洗净，控干水分备用；

3. 热锅凉油，放入姜末、葱花，爆香；

4. 放入鸡胗煸炒至变色，加入盐、鸡精调味；

5. 放入豆芽翻炒均匀，炒至豆芽变软入味即可。

鸡胗炒豆芽清淡爽口，营养丰富。

六、鸡胗炒蒜苗

食材：鸡胗一份（约400克）、蒜苗适量、葱姜适量、盐适量、生抽适量、香油适量、料酒适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，切成厚片或小段，加入料酒和盐腌制几分钟备用；

2. 蒜苗洗净切段，葱姜切末备用；

3. 热锅凉油，爆香葱姜末，加入鸡胗煸炒至变色；

4. 加入蒜苗，翻炒均匀，加入少量盐和生抽调味；

5. 最后加入香油，翻炒均匀即可。

鸡胗炒蒜苗香脆可口，美味不油腻。

七、鸡胗炒木耳

食材：鸡胗一份（约400克）、黑木耳适量、姜末适量、葱姜适量、盐适量、生抽适量、老抽适量、料酒适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，切成碎块，放入料酒中腌制15分钟；

2. 黑木耳清洗干净，泡发；

3. 热锅凉油，放入葱姜末和姜丝爆香；

4. 加入鸡胗煸炒至变色，加入盐、老抽、生抽调味；

5. 加入木耳翻炒均匀，火候稍微收一下汁，即可出锅。

鸡胗炒木耳，口感丰富，多种食材相互交融。

八、鸡胗炒莲藕

食材：鸡胗一份（约400克）、莲藕适量、姜适量、葱姜适量、盐适量、生抽适量、料酒适量、鸡精适量、色拉油适量。

做法：

1. 将鸡胗用清水冲洗干净，切成条状，加入盐和料酒拌匀腌制20分钟；

2. 莲藕去皮，洗净，切薄片备用；

3. 热锅凉油，爆香葱姜末和姜片；

4. 加入鸡胗煸炒至变色，加入盐、生抽、鸡精调味；

5. 加入莲藕翻炒均匀，炒至莲藕五分熟即可。

鸡胗炒莲藕，清爽不腻，肉脆莲脆相映成辉。
