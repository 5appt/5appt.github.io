海参的家常做法

更新时间：2023-05-19 20:02

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

不少于2000个字文章 海参是一种营养丰富的食材，含有丰富的蛋白质、钙、铁、硫酸软骨素等营养成分。海参可以烧炖、炒食、煲汤等多种方式食用，不仅美味可口，而且具有很好的保健作用。今天，我就带大家分享一些常见的海参的家常做法。

1. 海参炒鸡蛋

食材：海参、蛋、青葱、盐、生抽、植物油。 做法： 1. 海参洗净切成小块，放入清水中浸泡2小时，泡发后再用清水冲洗干净备用。 2. 鸡蛋打散备用，青葱切成葱花备用。 3. 热锅凉油，将鸡蛋倒入锅中煎制，煎至双面金黄后盛出备用。 4. 热锅凉油，倒入海参翻炒，待海参翻炒至变软之后，加入蛋液和盐、生抽翻炒均匀。 5. 最后加入青葱炒匀即可。

2. 海参扒菜心

食材：海参、菜心、鸡汤、盐、油。 做法： 1. 海参泡发后用清水洗净，菜心洗净切成小段备用。 2. 热锅凉油，将海参煎至两面微黄后盛出备用。 3. 将煎好的海参放入锅中，加入适量的鸡汤，焖煮2分钟。 4. 放入切好的菜心，继续焖煮2分钟，最后加入盐调味即可。

3. 红烧海参

食材：海参、花菇、葱姜蒜、酱油、糖、盐、料酒、香菜、植物油。 做法： 1. 将海参泡发洗净，花菇泡发洗净，葱姜蒜切碎备用。 2. 热锅凉油，倒入葱姜蒜爆香。 3. 加入泡发的海参和花菇翻炒，待海参微黄之后加入适量的酱油、糖、盐、料酒调味，翻炒均匀。 4. 倒入适量的水，焖煮15分钟左右，等汤汁收浓后撒上香菜即可。

4. 蒜泥海参

食材：海参、蒜头、香油、盐、白胡椒粉。 做法： 1. 海参泡发后用清水洗净备用，蒜头切碎备用。 2. 将泡发的海参放入锅中焯水煮5分钟，煮至海参软烂，去除表面的沙粒。 3. 将海参放入盘中，撒上盐和白胡椒粉。 4. 蒜头切碎，加入热油中爆香后淋在海参上，撒上一些香油即可。

5. 豆腐海参煲

食材：海参、嫩豆腐、葱姜蒜、香菇、鸡汤、酱油、盐、料酒、植物油。 做法： 1. 海参泡发后用清水洗净备用，嫩豆腐切成块状，葱姜蒜切碎备用，香菇泡发洗净备用。 2. 热锅凉油，将葱姜蒜爆香，放入海参和香菇翻炒。 3. 加入适量的鸡汤煮开，放入嫩豆腐、酱油、盐、料酒调味，煮开后小火焖煮25分钟左右。 4. 最后放入适量的香油调味，翻炒均匀即可。 以上就是我分享的几种海参的家常做法，相信大家已经掌握了。最后还要提醒大家，海参虽然营养丰富，但不易消化，故在食用时应适量，不能过量食用。
