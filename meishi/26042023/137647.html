山药粥的做法大全

更新时间：2023-04-26 13:02

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

山药粥是一种营养丰富、滋补身体的粥品，具有滋润肺阴、滋补元气、益肾养胃的效果。下面介绍几种不同口味的山药粥做法。

1. 山药桂圆红枣粥

材料：

山药100克、桂圆15~20个、红枣5颗、糯米50克、白糖适量。

做法：

1. 将糯米淘洗干净后，用清水浸泡2小时左右。

2. 山药去皮切块，桂圆、红枣洗净备用。

3. 将泡好的糯米过滤掉水分，加入适量的清水，开火煮开后转小火，煮至米粥变软糯，加入山药、桂圆、红枣继续煮烧。增加煮的时间，直到呈泥状即可。

4. 最后加入少许白糖调味即可。

2. 五谷杂粮山药粥

材料：

山药100克、五谷杂粮（糙米、小米、玉米、红豆等）适量、水适量。

做法：

1. 将五谷杂粮混合后清洗干净。

2. 山药去皮切块备用。

3. 将五谷杂粮用适量清水煮至软烂，放入山药块，继续煮15-20分钟左右。

4. 最后根据个人口味加入适量的糖即可。

3. 红豆山药粥

材料：

山药100克、红小豆50克、糯米50克、红枣5颗、盐适量。

做法：

1. 将红小豆清洗干净后泡水1小时，糯米也清洗干净，泡水20分钟。

2. 山药去皮，切块；红枣洗干净。

3. 把泡好的红小豆、糯米放入锅中，加适量清水，开启小火煮20分钟左右。

4. 加入山药块、红枣，继续煮开，最后加入适量盐调味，煮到粥浓汤稠即可。

4. 木瓜山药粥

材料：

山药100克、木瓜适量、珍珠米50克、水适量。

做法：

1. 珍珠米清洗干净后，浸泡15-30分钟。

2. 山药去皮洗干净，切成小块，木瓜去籽，切成小块。

3. 加入适量清水，将珍珠米煮沸，转小火煮20分钟。

4. 加入山药块、木瓜块，煮至山药、木瓜都软烂（约10分钟），最后加入少许白糖调味即可。

5. 核桃山药粥

材料：

山药100克、核桃30克、大米50克、红枣5个、水适量。

做法：

1. 将山药去皮洗干净，切成小块；大米洗净后泡水20分钟；红枣去核洗净。

2. 核桃将肉取出，打成粉。

3. 将大米放入锅里加适量清水，煮5分钟后，加入山药块、红枣，继续煮开。

4. 最后加入核桃粉，煮5分钟即可。

小提示：山药中含有的黏液质较多，所以在做山药粥时，最好淘洗干净，加入适量的清水，煮的时间不宜过久，免得粘稠度太高，影响口感。
