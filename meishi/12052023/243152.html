莴笋的做法大全家常菜

更新时间：2023-05-12 06:29

莴笋是一种非常营养的蔬菜，含有丰富的维生素C、膳食纤维以及钾等营养素。它的口感清脆爽口，可以用来制作汤、凉拌、炒菜等多种菜肴。下面就来介绍一些家常莴笋的做法。

一、莴笋汤 莴笋汤是一道营养丰富的汤品。材料需要莴笋、猪肉、枸杞、生姜、盐等。制作方法如下： 1. 将莴笋削皮切成小片，猪肉切成薄片备用； 2. 生姜切片，枸杞洗净备用； 3. 锅中加水，放入生姜后加入莴笋片和枸杞，大火烧开后转小火煮20分钟； 4. 加入猪肉片煮5分钟，加盐调味即可。

二、莴笋炒鸡蛋 这道菜做法简单，口味鲜美，既可当早餐也可当晚餐。材料需要莴笋、鸡蛋、盐、生姜等。具体做法如下： 1. 将莴笋去皮切成小段，生姜切成细末； 2. 鸡蛋打散，加入盐打匀备用； 3. 锅中放少许油，加入姜末炒香； 4. 加入莴笋翻炒均匀，加盐，翻炒至莴笋软烂； 5. 加入鸡蛋液，翻炒均匀即可。

三、莴笋烧鸡腿 这道菜是一道营养丰富的家常菜肴，制作方法如下： 1. 将鸡腿切成块，用剪刀将鸡肋骨处理干净； 2. 将鸡肉放入沸水中焯水； 3. 莴笋切成小段焯水，也可用开水泡软； 4. 焯好的鸡肉和莴笋放入砂锅中，加入适量的生姜、料酒、盐，加适量的水； 5. 大火烧开后转小火炖煮1个小时左右，加适量的味精即可。

四、莴笋炒牛肉 这道菜做法简单，需要的材料有莴笋、牛肉、豆瓣酱、盐等。具体制作方法如下： 1. 将牛肉切成薄片，加入盐、淀粉搅拌均匀，腌制10分钟； 2. 将莴笋处理干净后切成细条备用； 3. 锅中放少许油，加入豆瓣酱煸炒出香味后，加入牛肉快速翻炒至变色； 4. 加入莴笋翻炒均匀，加盐调味即可。

五、莴笋糯米饭 这道菜不仅口感绵软，营养丰富。具体制作方法如下： 1. 先将糯米泡发2- 3小时，控干水分备用； 2. 将莴笋洗净后切成细丝，放入碗中； 3. 在碗中加入适量的盐、料酒、鸡精、蒜末、油拌匀后，加入糯米拌匀备用； 4. 将蒸饭锅，放入糯米和莴笋的混合物，加水蒸熟即可。

六、莴笋炒豆腐 豆腐和莴笋搭配在一起，既是一道美味又是一道营养的菜肴。具体制作方法如下： 1. 蒜切末，莴笋切成小段，豆腐切成小块备用； 2. 锅中放油，加入蒜末煸炒出香味； 3. 加入莴笋翻炒均匀，加盐焖至软烂； 4. 最后加入豆腐翻炒1分钟即可。

七、莴笋烤肉串 这道菜会让人感到开胃，适合作为家庭聚餐的一道菜肴。制作方法如下： 1. 将猪肉切成小块，用盐、孜然粉、辣椒粉、蒜末搅拌均匀； 2. 将莴笋切成小段，用水焯烫一下； 3. 烤肉串之前要将莴笋和肉串穿起来； 4. 在肉串两面涂上蜂蜜，然后放到烤箱中进行烤制即可。

八、莴笋烤鳕鱼 这道菜口感鲜美，适合作为一道健康美味的薄荷料理。具体做法如下： 1. 鳕鱼切成大块，加盐腌制10分钟； 2. 莴笋、红椒分别切成小段备用； 3. 将鳕鱼和莴笋、红椒混合在一起抹上蒜泥、橄榄油，放进烤箱烤12分钟即可。

九、莴笋烤鸡翅 这道菜味道鲜美，口感清香，做法简单。制作方法如下： 1. 将鸡翅中分割成两半备用； 2. 将莴笋洗净，去皮切成小段； 3. 在鸡翅上撒上盐、黑胡椒，将莴笋放在烤盘上； 4. 烤盘放入预热至200度的烤箱中烤15分钟，翻转鸡翅后再烤10分钟，即可出炉。

十、莴笋烤三文鱼 这道烤三文鱼搭配莴笋非常清香爽口，步骤如下： 1. 将三文鱼用盐、橄榄油、黑胡椒粉腌制1小时； 2. 将莴笋去皮、切成小段，放在烤盘上； 3. 烤盘放进预热至200度的烤箱中烤10分钟，取出后放上三文鱼，再烤10分钟即可。

总结： 莴笋这种蔬菜口感清脆，同时营养丰富，可以做成多种美味菜肴。以上介绍的这几种莴笋菜肴方法简单易学，大家可以根据自己的口味喜好来调整配料。不妨试试这些莴笋做法吧！
