炒卷心菜的做法大全！让你做出美味无比的卷心菜

更新时间：2023-04-19 06:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

卷心菜是一种非常常见的蔬菜，它富含丰富的维生素C、钾、膳食纤维等多种营养物质，具有清热解毒、解油腻、开胃健脾等多种功效。炒卷心菜是一道非常简单好做又美味的菜肴，下面就为大家介绍几款炒卷心菜的做法，让你轻松做出美味无比的卷心菜。

一、蒜蓉炒卷心菜

材料：卷心菜、大蒜、盐、油

做法：

1. 将卷心菜清洗干净，切成小段备用；

2. 取适量的大蒜切成蒜蓉，备用；

3. 在锅中倒入适量的油，加热后倒入蒜蓉煸炒出香味；

4. 加入卷心菜，不断翻炒，炒出卷心菜的香气；

5. 最后加入盐调味即可。

二、卷心菜炒腊肉

材料：卷心菜、腊肉、油、盐、葱姜蒜、料酒

做法：

1. 将卷心菜洗净后切成段备用；

2. 葱姜蒜切成末，备用；

3. 将腊肉切成小丁，入油锅煸炒出香味；

4. 加入葱姜蒜末，煸炒出香味；

5. 加入卷心菜，翻炒均匀后倒入料酒焯水；

6. 加盐调味，翻炒均匀即可。

三、醋溜卷心菜丝

材料：卷心菜、大蒜、醋、盐、油、糖、葱

做法：

1. 将卷心菜去蒂部分，切成细丝备用；

2. 取适量的大蒜切成蒜蓉，备用；

3. 热锅倒入油，加入大蒜爆香后倒入卷心菜爆炒；

4. 顺次加入醋、盐、糖，不断翻炒均匀；

5. 最后撒上一些葱花点缀即可。

四、红烧卷心菜

材料：卷心菜、瘦肉、姜、葱、盐、料酒、豆瓣酱、酱油、糖、味精

做法：

1. 热锅凉油，放入瘦肉煎至金黄色后捞起备用；

2. 锅中留少许底油，放入姜、葱爆香；

3. 加入豆瓣酱，煸炒出红油后加入卷心菜；

4. 加入适量的盐、糖、酱油、料酒，白糖，翻炒均匀；

5. 加水没过卷心菜，大火煮开，转小火焖煮至卷心菜熟透，加入味精调味即可。

五、香菇卷心菜卷

材料：卷心菜、香菇、肉末、鸡蛋、盐、姜末、植物油

做法：

1. 热锅冷油，加入姜末和肉末略炒；

2. 放入香菇略炒，盐调味；

3. 把卷心菜叶切去硬脊部分，用刀刮一下表面，涂上鸡蛋液；

4. 向卷心菜叶的底部放入香菇和肉末，逐个卷起；

5. 将卷心菜卷放入鸡蛋液中裹上一层蛋液；

6. 热锅烧油，用中小火，将卷心菜卷放入油锅中煎炸至金黄色即可。

以上几款炒卷心菜的做法不仅简单易学，而且口感鲜美，可以满足不同人群的口味需求。快学起来，做出美味的卷心菜吧！
