家常素菜100道，让你轻松变身美食达人

更新时间：2023-04-30 08:43

素菜不仅美味健康，而且易于制作。无论是快速晚餐还是招待客人，素菜都是不错的选择。今天，小编为大家准备了100道家常素菜食谱，不仅简单易做，而且尝到的滋味绝对超出你的预期。

1. 松仁炒西葫芦

材料：西葫芦、松仁、盐、姜、葱、食用油 做法： 1. 西葫芦切成小段，放入盐水中焯水，捞出备用。 2. 热锅凉油，爆香姜葱末，加入松仁煸炒。 3. 加入焯水好的西葫芦，翻炒均匀。 4.出锅前撒上少许盐即可。

2. 红烧茄子

材料：茄子、葱、姜、蒜、红椒、豆瓣酱、生抽、老抽、糖、醋、清水、食用油 做法： 1. 茄子切成小块，加盐腌制。 2. 葱姜蒜切末，红椒切成细丝。 3. 热锅凉油，爆香葱姜蒜。 4. 加入豆瓣酱和红椒丝炒香。 5. 加入茄子翻炒2-3分钟。 6. 加入生抽、老抽、糖、醋、清水烧开，转小火慢炖20分钟即可。

3. 土豆烧牛肉

材料：牛肉、土豆、胡萝卜、豆腐干、葱、姜、大蒜、红椒、干辣椒、八角、桂皮、香叶、料酒、生抽、老抽、盐、香油、食用油 做法： 1. 牛肉切块焯水捞出备用，土豆切成大块。 2. 葱、姜、大蒜切末，红椒切成丝。 3. 热锅凉油，爆香葱姜大蒜，放入牛肉煸炒。 4. 加盐、料酒、生抽、老抽煸炒均匀。 5. 加入八角、桂皮、香叶、干辣椒、红椒翻炒。 6. 加入土豆和胡萝卜，加适量的水烧开。 7. 转小火慢炖40分钟。 8. 最后放入豆腐干和香油翻炒，即可出锅。

4. 青椒炒鸡蛋

材料：鸡蛋、青椒、盐、生抽、食用油 做法： 1. 鸡蛋打散，青椒切成丝。 2. 热锅凉油，倒入鸡蛋液，快速翻炒。 3. 炒至鸡蛋凝固，捞出备用。 4. 热锅凉油，爆香青椒丝。 5. 加入炒好的鸡蛋，加适量盐和生抽翻炒均匀即可出锅。

5. 番茄炒蛋

材料：鸡蛋、番茄、盐、姜、葱、食用油 做法： 1. 鸡蛋打散，加适量盐，搅拌均匀。 2. 番茄去皮切小块，姜葱切末。 3. 热锅凉油，放入姜葱煸香。 4. 加入炒散的鸡蛋，快速翻炒。 5. 加入切好的番茄翻炒均匀。 6. 最后撒上少许盐即可出锅。

6. 炒三丝

材料：胡萝卜、木耳、黄瓜、姜、蒜、葱、盐、生抽、食用油 做法： 1. 胡萝卜切成丝备用，木耳泡软备用，黄瓜切丝备用。 2. 葱姜蒜切末备用。 3. 热锅凉油，放入葱姜蒜煸香。 4. 加入胡萝卜丝翻炒。 5. 加入木耳和黄瓜翻炒均匀。 6. 最后撒上少许盐和生抽即可出锅。

7. 大葱花炒蛋

材料：鸡蛋、大葱、盐、生抽、食用油 做法： 1. 鸡蛋打散加盐。 2. 大葱切成葱花备用。 3. 热锅凉油，将鸡蛋液倒入锅中，快速翻炒出成鸡蛋块备用。 4. 热锅凉油，爆香大葱花。 5. 将炒好的鸡蛋块加入，加适量的盐和生抽翻炒均匀即可出锅。

8. 烤南瓜

材料：南瓜、盐、黑胡椒、橄榄油、蜂蜜 做法： 1. 将南瓜削皮去瓤，切成块状。 2. 将南瓜块放入盐水中煮约5分钟至软。 3. 将南瓜块放入烤盘中。 4. 加适量盐和黑胡椒，淋上橄榄油和蜂蜜，均匀涂抹在南瓜块上。 5. 放入已经预热好的烤箱中，以200度烤约25分钟即可。

9. 青瓜汤

材料：青瓜、盐、鸡精、清水 做法： 1. 将青瓜切成小块。 2. 热锅凉油，加入青瓜小块翻炒2分钟。 3. 加入清水煮开，加适量盐和鸡精。 4. 小火煮30分钟即可。

10. 炒素菜

材料：洋葱、红椒、青椒、黄瓜、胡萝卜、盐、生抽、食用油 做法： 1. 将洋葱、红椒、青椒、黄瓜和胡萝卜都切成小块。 2. 热锅凉油，加入洋葱小块煸炒至香。 3. 将全部蔬菜倒入锅中轻易翻炒。 4. 最后加适量盐和生抽调味。 5. 出锅前再次翻炒均匀即可。

这些家常素菜食谱可以不同时间、不同场合制作，既可以做正餐，也可以做夜宵。快来试试吧，相信你也可以制作出美味的素菜佳肴。
