炒鱿鱼的做法大全

更新时间：2023-04-30 13:16

炒鱿鱼是一道非常受欢迎的海鲜菜肴，风味独特，口感鲜美。炒鱿鱼的做法有很多种，下面我们来介绍一些比较常见的炒鱿鱼做法。

炒鱿鱼的基本做法

1. 准备材料：鱿鱼、葱姜蒜、辣椒、胡萝卜、青椒等。

2. 切好鱿鱼，加入盐、料酒、生粉腌制十分钟。

3. 热锅加油，下葱姜蒜爆香后放入鱿鱼煸炒。

4. 加入青椒、胡萝卜、辣椒继续翻炒，最后加入调味料和香菜即可。

蒜蓉炒鱿鱼

1. 准备材料：鱿鱼、蒜、辣椒、生姜。

2. 将鱿鱼洗净，处理好，切成薄片，加入盐、料酒、胡椒粉拌匀。

3. 起锅加油，放入蒜末、辣椒、姜末煸炒出香味。

4. 将腌好的鱿鱼放进锅里，迅速翻炒。

5. 炒到鱿鱼稍微卷曲，出锅前加入盐、味精调味即可。

酱爆炒鱿鱼

1. 准备材料：鱿鱼、葱姜蒜、辣椒、盐、酱油。

2. 将鱿鱼切成块，加入盐、料酒、白胡椒粉腌制十分钟。

3. 起锅热油，放入葱姜蒜煸炒出香味。

4. 加入调料翻炒，然后放入鱿鱼炒至稍微卷曲。

5. 最后加入酱油调味，翻炒均匀，出锅前撒上小葱末即可。

葱爆炒鱿鱼

1. 准备材料：鱿鱼、葱、姜、蒜、料酒、酱油、盐。

2. 将鱿鱼处理好，切成薄片，加入盐、料酒拌匀腌制十分钟。

3. 起锅加油，放入葱姜蒜爆香，然后放入处理好的鱿鱼快速翻炒。

4. 加入酱油调味，撒入切好的葱花拌匀即可。

干锅炒鱿鱼

1. 准备材料：鱿鱼、葱姜蒜、干辣椒、花椒、醋、盐。

2. 将鱿鱼切成块，加入盐、料酒、胡椒擦拌均匀腌制十分钟。

3. 起锅加油，放入葱姜蒜、干辣椒、花椒爆香。

4. 将腌制好的鱿鱼加入锅里翻炒，等鱿鱼颜色变浅变红时加入醋和盐，继续翻炒均匀即可。

酸菜炒鱿鱼

1. 准备材料：鱿鱼、酸菜、葱姜、盐、鸡精。

2. 鱿鱼处理好，切成片状，加入盐、料酒腌制半小时。

3. 起锅热油，放入葱姜爆香后加入酸菜翻炒均匀。

4. 加入腌制好的鱿鱼炒至变色即可。

5. 最后加入鸡精调味，翻炒均匀出锅即可。

清蒸鱿鱼

1. 准备材料：鱿鱼、大葱、姜丝、料酒、盐、醋。

2. 鱿鱼处理好，切成镂空的片状，撒上料酒、盐、姜丝腌制十分钟。

3. 将腌好的鱿鱼码放于碗里，放入葱段，加入清水略过鱿鱼部分，不要淹没鱿鱼。

4. 大火蒸十分钟左右，取出鱿鱼，倒掉水，撒上葱花和醋即可。

以上就是关于炒鱿鱼的做法大全，不同做法口感各异，喜欢吃鱿鱼的朋友可以上手一试。
