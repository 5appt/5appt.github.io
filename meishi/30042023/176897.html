杏鲍菇的做法大全

更新时间：2023-04-30 08:43

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

杏鲍菇是一种常见的菌类，营养丰富，口感鲜美，又称为春菇。它含有多种氨基酸、蛋白质、维生素和矿物质，对人体健康有很多好处。杏鲍菇的烹饪方法也很多样，可以炒、煮、蒸、炖等，下面介绍几种常见的做法。

1. 杏鲍菇炒肉片

材料：杏鲍菇、猪肉片、洋葱、蒜

调料：盐、酱油、料酒、糖、生抽、淀粉、胡椒粉

做法：

1. 杏鲍菇撕成小块，猪肉片入料酒、盐、胡椒粉拌匀

2. 热锅倒油，炒香蒜，爆炒洋葱和猪肉片

3. 加入杏鲍菇，炒至熟软，调入盐、糖、酱油、生抽，翻炒均匀

4. 加入少许水，勾芡后出锅，鲜香可口。

2. 杏鲍菇凉拌

材料：杏鲍菇、胡萝卜、黄瓜、生菜、香菜

调料：盐、酱油、醋、蒜、辣椒油、花椒粉、芝麻酱

做法：

1. 杏鲍菇用清水洗净，横均切成丁

2. 将胡萝卜、黄瓜切成丝，洗净生菜叶

3. 将杏鲍菇、胡萝卜、黄瓜、生菜放在碗里，撒上少许盐，腌制5分钟

4. 将蒜捣成蒜泥，加入酱油、醋、花椒粉、辣椒油、芝麻酱，搅拌均匀

5. 将调料浇在杏鲍菇和蔬菜上，加入香菜，拌匀即可。

3. 杏鲍菇蒸鸡蛋

材料：杏鲍菇、鸡蛋、姜、葱

调料：盐、鸡精、料酒、酱油、香油

做法：

1. 把杏鲍菇洗干净，挑去杂质，切成薄片

2. 鸡蛋打入碗中，加入姜末、葱花、盐、鸡精、料酒、酱油、香油，搅拌均匀

3. 将蒸碗刷上一层油，把杏鲍菇铺在碗底

4. 把鸡蛋液倒入碗中，蒸10分钟，用筷子戳一下鸡蛋看看是否凝固

5. 完成后取出，淋上少许香油，撒上葱花即可。

4. 杏鲍菇炖鸡肉

材料：杏鲍菇、鸡肉、胡萝卜、土豆、洋葱、姜、蒜

调料：盐、鸡精、料酒、酱油、八角、桂皮、香叶、水

做法：

1. 将鸡肉切成小块，放入开水中焯水，捞出备用

2. 杏鲍菇、土豆、胡萝卜、洋葱、姜、蒜切成块备用

3. 将炖盅刷上少许油，倒入鸡肉块、杏鲍菇、胡萝卜、土豆、洋葱、姜、蒜

4. 加入八角、桂皮、香叶、盐、鸡精、料酒、酱油，加入适量水

5. 制好盖子，放入蒸锅中，大火煮开后转为小火炖45分钟，成品即可。

5. 杏鲍菇煮粥

材料：杏鲍菇、大米、姜、葱、油、盐

做法：

1. 大米淘洗干净，放入清水中浸泡30分钟，捞出备用

2. 杏鲍菇洗净，切成小块

3. 锅中加少许油，烧热后放入葱、姜爆香

4. 倒入大米和杏鲍菇，加适量清水，大火烧开后转为小火煮熟

5. 加入适量盐，煮5分钟即可。

以上是杏鲍菇的几种常见做法，食材简单易得，做法简单易学，美味可口，值得一试。
