秋季养生食谱家常菜

更新时间：2023-04-30 13:16

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

秋天是一个收获的季节，也是一个充满变化的季节。气温逐渐降低，万物凋零，但是，这并不妨碍我们的好心情和美好生活。秋季正是我们养生和保健的重要时期，合理饮食更能帮助我们提高身体免疫力、增强身体健康。下面给大家介绍几道适合秋季养生的家常菜。

1. 煲汤养生菜

秋季气温逐渐变冷，喝点汤可以温暖胃，提高身体免疫力。这道煲汤养生菜汤汁浓郁、清香可口，营养丰富。

材料：鸡胸肉150克，西兰花、红枣、干贝、白菜、莲藕、枸杞、姜片、大葱适量，盐、鸡精适量。

做法：

1. 鸡胸肉煮熟切丝，红枣、干贝泡发洗净，西兰花、莲藕、白菜洗净备好。

2. 水烧开后把洗净的菜和食材放进去，煲1 - 1.5小时。

3. 汤熬开后加入适量的盐和鸡精，煮几分钟即可。

2. 低脂养生鸡

这道低脂养生鸡是用红枣、花生等家常食材搭配炖鸡，口感清香，还可以养生健体。

材料：鸡腿肉、姜、花生仁、红枣、葱、盐、白胡椒粉、料酒、鸡汤适量。

做法：

1. 先将鸡腿洗净，入沸水中煮5分钟，去除血水，并漂洗干净。

2. 葱姜切末，花生仁煸炒至香味出来。

3. 锅中放入少许油，烧热后将葱姜末煸炒，然后放入鸡块和花生翻炒。

4. 加入鸡汤、红枣、料酒，大火煮开后调至小火，煨炖30分钟，加适量盐和白胡椒粉即可。

3. 养生鲜虾炒根菜

虾含有丰富的优质蛋白质和钙质，还具有降脂效果，非常适合秋季饮食。

材料：鲜虾、洋葱、胡萝卜、土豆、蒜、油、盐、鸡精、葱花适量。

做法：

1. 首先虾去壳洗净，洋葱和蒜切小块，土豆、胡萝卜切片或小块。

2. 热锅凉油，放入蒜葱和虾仁爆香，然后加入洋葱、土豆、胡萝卜翻炒，炒至变软。

3. 加入水，盖上锅盖煮5分钟，直到土豆煮熟即可

4. 最后加适量的盐、鸡精，撒上葱花即可。

4. 养生番茄鸡蛋面

秋季天气干燥，多吃含有植物纤维的食品可以润肠通便。这道番茄鸡蛋面中含有养生又营养的蔬菜和优质的蛋白质，非常适合秋季食用。

材料：鸡蛋、小番茄、芹菜、草菇、绿豆芽、葱、姜、面条、油、盐、鸡精适量。

做法：

1. 热锅凉油，放入葱姜，然后加入芹菜、草菇、绿豆芽翻炒2分钟。

2. 加入清水，放入面条煮至熟后捞出备用。

3. 锅中留少许油，加入打散的鸡蛋，煮至金黄色出锅备用。

4. 锅中再倒入少许油，放入小番茄快速翻炒，再把之前的煮好的面条、鸡蛋混合在一起，加入适量的盐和鸡精翻炒均匀即可。

总之，秋季养生食谱需要综合考虑个人体质和各种食材的营养价值，才能达到最佳的养生效果。上述几道精美而营养丰富的家常菜绝对是秋季的养生良方，各位朋友可以尝试一下。
