凉拌菜的做法大全

更新时间：2023-05-03 10:40

凉拌是一道清爽可口的菜肴，不仅易于制作，而且味道鲜美，受到了大众的喜爱。不同的蔬菜材料和酱料组合，可以制作出多种口感和味道的凉拌菜肴。下面介绍一些凉拌菜的做法大全，让您可以轻松在家制作出口感鲜美的凉拌菜肴。

1. 韭菜花凉拌

材料：

韭菜花300g、蒜1个、生姜1小块、花生米适量、醋少许、食盐少许、香油适量。

制作过程：

1. 将韭菜花洗净，焯水后捞起沥干水分；

2. 蒜、生姜切末；

3. 热油锅，下入蒜、姜末爆香；

4. 加入花生米煸炒至金黄，取出备用；

5. 韭菜花加入适量的盐、醋拌匀；

6. 倒入蒜姜、花生米和适量香油，拌匀即可食用。

2. 黄瓜凉拌

材料：

黄瓜1根、蒜末适量、生姜末适量、醋少许、盐适量、糖适量、香油适量。

制作过程：

1. 黄瓜洗净，去皮切成条状；

2. 热锅倒入适量油，加入蒜末和生姜末炒香，加入盐、糖和醋拌匀；

3. 将黄瓜条加入蒜姜酱中，拌匀，加入香油即可。

3. 豆腐皮凉拌

材料：

豆腐皮2张、葱花适量、蒜末适量、红油豆瓣酱适量、醋适量、盐适量、香油适量。

制作过程：

1. 豆腐皮焯水后切成丝状；

2. 葱花和蒜末炒香，加入豆瓣酱后继续炒香；

3. 加入适量醋和盐拌匀，然后加入豆腐皮丝拌匀；

4. 加入适量的香油即可食用。

4. 拌凉粉

材料：

凉粉一袋、胡萝卜适量、黄瓜适量、葱花适量、醋适量、盐适量、香油适量。

制作过程：

1. 凉粉焯水后切成条状；

2. 胡萝卜、黄瓜切成丝状，葱花切碎；

3. 将凉粉和蔬菜丝混合，加入适量醋和盐拌匀；

4. 加入适量香油即可食用。

5. 凉拌蒟蒻粉

材料：

蒟蒻粉一袋、黄瓜适量、胡萝卜适量、小黄瓜适量、醋适量、盐适量、辣椒油适量。

制作过程：

1. 蒟蒻粉加水煮至熟，捞起沥干水分；

2. 黄瓜、胡萝卜、小黄瓜切成丝状；

3. 将蒟蒻粉和蔬菜丝混合，加入适量的醋和盐拌匀；

4. 加入适量的辣椒油即可食用。

6. 香菜拌豆腐干

材料：

豆腐干适量、香菜适量、蒜末适量、酱油适量、盐适量。

制作过程：

1. 豆腐干切成小块；

2. 香菜切末；

3. 将豆腐干、香菜和蒜末混合，加入适量的酱油和盐拌匀。

7. 凉拌茶树菇

材料：

茶树菇适量、葱姜蒜适量、醋适量、盐适量、香油适量。

制作过程：

3. 将茶树菇加入葱姜蒜中，加入盐和醋拌匀；

4. 加入香油即可食用。

8. 凉拌莲藕丝

12. 酸辣海蜇丝

15. 凉拌木耳
