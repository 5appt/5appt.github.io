年夜饭菜谱：让你的新年大餐更加丰盛

更新时间：2023-05-03 20:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

春节是中国最重要的传统节日，也是家庭团聚的时刻。除了重视亲情和友情，年夜饭作为年节的重要食物之一也被人们所关注。一桌丰盛的年夜饭，在祝福和期盼中吃过去，是我们一年中最幸福的时刻。然而，如何准备一次丰富且有意思的年夜饭呢？本篇文章将精选不少于18道菜品，详细介绍制作步骤和口感特点，为你的新年大餐锦上添花。

鸿运当头八宝饭

材料：

糯米、糖、干虾米、火腿、枸杞、栗子、莲子、红枣、花生、葡萄干。

制作方法：

1. 用清水泡发糯米，大约4小时后将其蒸熟，备用；

2. 将莲子和栗子烧熟，去皮备用；

3. 将干虾米和火腿切成末，用油煸炒一下，加入适量糖拌匀；

4. 将所有材料混合在一起，再放入蒸锅中蒸20分钟即可。

这道八宝饭十分特别，不仅颜色艳丽，而且味道丰富。煮熟的糯米粘稠可口，里面的各种材料更是五味俱全。

海鲜大拼盘

材料：

龙虾、海螺、扇贝、鱿鱼、鲍鱼、翅尖刺参、蒜蓉、香菜。

制作方法：

1. 将各种海鲜洗净后去壳，待用；

2. 捣碎几瓣蒜，热锅凉油，加入蒜泥；

3. 将所有的海鲜放入蒜油中爆炒翻炒几分钟；

4. 准备几片香菜，放于海鲜的上面，即可食用。

一份美味的海鲜大拼盘，让你的年夜饭更加华丽豪放。龙虾、海螺、扇贝都是海鲜中的佳品，盐煮后更是口感令人陶醉，绝对让你回味无穷，且让你更加有面子。

百花缤纷荷包蛋

材料：

鸡蛋、芹菜、胡萝卜、豌豆、玉米泥、面粉、油、盐、酱油。

制作方法：

1. 将芹菜和胡萝卜切成小块，焯水备用；

2. 打入适量的鸡蛋，拌匀，放少量盐和酱油，再加入玉米泥和焯水的胡萝卜、芹菜和豌豆，拌匀；

3. 在平底锅中少量油，将面粉过筛后放置在油中，用铲子翻动，煸炒至成深褐色的糊状体；

4. 将鸡蛋液倒入锅中，转小火煎至面皮上浮，煎至金黄色即可。

这道荷包蛋，色彩斑斓，口感鲜美。蔬菜中的胡萝卜、芹菜和豌豆增加了口感和美观程度，红绿相间，吃起来更有食欲。荷包蛋的面皮上糊糊的风味也十分独特。

黄金脆皮鸭

材料：

鸭子、料酒、盐、糖、蜂蜜、老抽、酱油、细辣椒。

制作方法：

1. 洗净鸭子，切成块，加入料酒、盐、糖、老抽、酱油腌制3个小时，备用；

2. 烤肉用的烤箱预热，鸭肉块依次码放在烤盘上；

3. 在鸭子上刷上一层清蜜，撒上细磨的辣椒，放入预热好的烤箱中，以200度的温度烤制40分钟。

这道黄金脆皮鸭，美味可口，皮脆肉嫩。蜜糖的涂抹让皮肉外焦里嫩，细辣椒的刺激却增加了汁水的浓艳。鸭身的大小形状可以根据家人数量定制。

五彩炒饭

材料：

米饭、鸡蛋、胡萝卜、青豆、玉米粒、墨鱼丝、黑胡椒粉、盐、酱油。

制作方法：

1. 把蛋打散，热锅倒油开始煎蛋饼，备用；

2. 将胡萝卜、玉米粒和青豆焯水备用；

3. 油热后放入煮熟的米饭，加入盐和酱油，翻炒至米饭散开；

4. 加入之前煎熟的蛋饼和焯水的胡萝卜、青豆和玉米粒，继续翻炒；

5. 最后放入墨鱼丝，撒上适量的黑胡椒粉进行翻炒，收均匀出锅即可。

这道五彩炒饭，色彩艳丽，口感绝妙。米饭和各种蔬菜的混合吃起来丝丝入味，再加上墨鱼丝的咸香味，也是炒饭爱好者的首选。

琳琅满目造型水果盘

材料：

草莓、橙子、猕猴桃、柚子、葡萄等水果。

制作方法：

1. 水果洗净，在安静干燥处晾干；

2. 把水果切成不同的形状，例如桔子切成片状，葡萄切成串状，草莓整颗摆放；

3. 把准备好的水果摆成不同的形态，可以是菊花状、心形、暴风雨等等，把自己的创意充分展现；

4. 最好使用大托盘，把摆好的水果放在盘子当中，旁边配上一些糖果即可。

这份造型水果盘，让新年的气氛加倍。水果如红艳的草莓、黄绿相间的猕猴桃、柚子的苦涩和橙子的酸甜，自成风味，非常好看且美味可口，特别适合放到年夜饭中。

幸福芝士焗贝壳

材料：

贝壳、洋葱、青椒、红椒、鲜奶油、芝士等。

制作方法：

1. 将贝壳洗净，摆放在烤盘上备用；

2. 将青椒、红椒、洋葱切成细丝，备用；

3. 在碗中添加鲜奶油、芝士渣，搅拌均匀，加适量盐调味；

4. 将调好味的鲜奶油和芝士倒在贝壳上，放上青椒、红椒和洋葱条，放入预热好的烤箱中，以200度的温度烤制15分钟。

这道幸福芝士焗贝壳可以说是色香味俱佳。鲜奶油和芝士的混合使得整个餐品口感之美另人陶醉。同时，烤焦的边缘和浅金色的表面也是美味的标志特征。

鲜美红烧牛肉

材料：

牛肉、土豆、胡萝卜、洋葱、大料、料酒、生块糖、老抽、盐、生姜、葱姜。

制作方法：

1. 牛肉、土豆和胡萝卜切成小块备用；

2. 锅中放油，放糖熔化，炒至呈金黄色，加入肉块煸炒；

3. 倒入料酒、老抽，加适量水稍稍过肉块，放入洋葱和姜片和大料煮开，撇去浮沫，盖上锅盖焖20分钟；

4. 等待肉烧至微烂，加入土豆和胡萝卜，煮至土豆熟透，放入盐调味即可。

这道红烧牛肉，肉质鲜美，色泽红亮。大火慢炖，让牛肉的汁水得到有效地保留，慢慢熬煮出肉质的精华，加上土豆和胡萝卜，让这道菜品的口感更加丰富多彩。

瑶柱虾仁烩时蔬

材料：

瑶柱、鲜虾仁、胡萝卜、西兰花、香菇、白芝麻。

制作方法：

1. 虾仁、瑶柱略稍微焯一下水后撇去浮沫；

2. 香菇、胡萝卜、西兰花均洗净备用；

3. 锅热后用辣椒、姜片爆香，加点鸡汤，将虾仁、瑶柱炖上5分钟取出，留汁备用；

4. 热锅加油，放入胡萝卜和西兰花，快炒几下，倒入虾仁瑶柱汤烧开，加入适量盐调味，撒上白芝麻即可。

这道瑶柱虾仁烩时蔬，只看颜色就已赏心悦目。配上香菇、胡萝卜和西兰花这些常见的时蔬，让味觉上得到新的体验，缸炉火甚好，令人大呼过瘾。
