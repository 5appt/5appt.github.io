点心做法大全：手把手教你制作十款美味点心P

日期：2023-04-16 栏目：美食常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

点心，是指茶点，一种小吃和饮品组合，是中华文化中的一个重要部分。在中国茶文化中，点心是不可或缺的，早茶晚饭都要配上点心。今天我们来分享十款美味点心的制作方法，让您在家也能品尝到餐厅级别的点心。 一、葡式蛋挞 所需材料： 鸡蛋 3个 淡奶油 150克 牛奶 100克 细砂糖 80克 低筋面粉 15克 蔓越莓干 30克 糖粉 适量 动物性酥油皮 需要410克(可购买现成的冷冻酥皮) 制作步骤： 1. 将鸡蛋打入碗中，打散，加入细砂糖，搅拌至完全融合。 2. 加入淡奶油和牛奶，混合均匀。 3. 筛入低筋面粉，搅拌均匀，过筛备用。 4. 将蔓越莓干洗净晾干，加入蛋液中，拌匀。 5. 将酥油皮切成适合的大小，放入蛋挞模具中。 6. 向蛋挞模具中注入蛋液，注至9分满即可。 7. 烤箱预热至200度，将蛋挞放入烤箱中，烘焙18分钟左右。 8. 取出烤好的葡式蛋挞，撒上糖粉即可食用。 二、糯米鸡 所需材料： 糯米 300克 鸡腿肉 2只 冬菇 6朵 紫菜 适量 蛋黄 1个 酱油 适量 盐 适量 制作步骤： 1. 鸡腿肉去骨，切成小块，加入酱油腌制10分钟。 2. 将冬菇用温水泡软，切成小块。 3. 糯米洗净后，加入适量清水浸泡1个小时。 4. 沥干糯米之后，加入少量的盐和腌好的鸡肉，调匀。 5. 取一块糯米，搓成圆球状，然后把鸡肉和冬菇放入其中。 6. 用紫菜包裹住糯米球，再用蒸锅蒸20分钟。 7. 鸡肉糯米饭蒸好之后，抹上蛋黄，再放入烤箱中烤5分钟，就可以做出脆皮糯米鸡了。 三、芝士汉堡 所需材料： 细砂糖 15克 盐 5克 鸡蛋 1个 奶粉 15克 高筋面粉 250克 牛奶 100克 酵母 3克 橄榄油 30克 芝士 适量 制作步骤： 1. 把牛奶、细砂糖、鸡蛋、盐、奶粉和面粉混合搅拌。 2. 慢慢地把酵母加入并继续搅拌，逐渐加入橄榄油，揉成面团。 3. 面团揉好后，盖上湿布醒发1.5小时左右，至面团变成两倍大。 4. 将面团揉至光滑，分成小面团4份，搓圆。 5. 将小圆面团放入烤箱中的烤盘上，再把芝士碎放上去。 6. 在面团上再盖上一层小圆面团，压平，按成汉堡的样子，表面再撒上芝士碎。 7. 放入预热至200度的烤箱中烤10分钟即可。 8. 烤好的芝士汉堡可以直接当早餐或下午茶吃。 四、草莓布丁 所需材料： 草莓 250克 牛奶 200毫升 淡奶油 100克 明胶粉 15克 细砂糖 75克 制作步骤： 1. 将草莓切成小块备用。 2. 将细砂糖和明胶粉混合在一起备用。 3. 把牛奶烧开后，转小火慢慢加入糖和明胶粉，不断搅拌，直到糖和明胶粉完全融化，再加入草莓。 4. 拌匀后，水浴加热至草莓也完全融化。 5. 将淡奶油打成浓稠状。 6. 把淡奶油和草莓液混合均匀，倒入布丁模具中。 7. 将布丁放入冰箱中，冷藏4小时左右。 8. 取出布丁，倒扣在盘子上，淋上草莓酱即可。 五、蜜汁凉皮 所需材料： 凉皮 1袋 蜂蜜 3勺 玉米淀粉 1勺 鸡精 适量 大葱 1段 黑芝麻 适量 鸡蛋清 1个 制作步骤： 1. 将凉皮切成小块，加入热水中焯熟，捞出控水后放入凉水中泡制备用。 2. 将蜂蜜、玉米淀粉、鸡精调匀，备用。 3. 将凉皮放入碗中，加入蜜汁，拌匀后待用。 4. 大葱切成末，放入锅中煸炒至出香味，加水稍微煮一下。 5. 捞出葱末后，将煮好的水放入碗中，加入一些鸡精和鸡蛋清，拌匀成蛋花水。 6. 取凉皮放入碗中，淋上蛋花水，再撒上黑芝麻，就可以食用了。 六、咖喱鸡肉三明治 所需材料： 烤面包片 4片 鸡腿肉 1只 胡萝卜 1根 大葱 1根 洋葱 1个 咖喱粉 2勺 盐 适量 胡椒粉 适量 橄榄油 适量 制作步骤： 1. 将鸡腿肉洗净，切成小块，加入盐、胡椒粉搅拌均匀。 2. 热锅倒入橄榄油，放入鸡肉煎至两面金黄，然后取出备用。 3. 用同样的方法把胡萝卜、大葱和洋葱切成小块，放入锅中翻炒。 4. 加入适量的咖喱粉，继续翻炒。 5. 将煮好的蔬菜和鸡肉混合，搭配烤面包片即可。 七、奇异果泡芙 所需材料： 无盐黄油 100克 中筋面粉 80克 鸡蛋 3个 水 160毫升 盐 1克 糖 8克 奇异果 2个 淡奶油 200克 制作步骤： 1. 黄油放入锅中融化，加入160毫升的水，加入盐和糖，煮热后加入中筋面粉，用中小火不断搅拌，直到面团形成，取下备用。 2. 面团稍微降温后，逐个加入鸡蛋，不断搅打，直到鸡蛋全部打入面团中。 3. 将面团装入裱花袋中，挤成小球型状，放入烤盘中。 4. 放入预热至200度的烤箱中烤25分钟左右，至泡芙表面金黄即可。 5. 奇异果去皮、去核，倒入料理机中搅成泥，然后加入淡奶油搅打成奇异果奶油。 6. 将制作好的泡芙切半，加入奇异果奶油即可。 八、珍珠鸡蛋糕 所需材料： 鸡蛋 2个 淡奶油 80克 细砂糖 50克 中筋面粉 80克 珍珠粉 适量 制作步骤： 1. 将鸡蛋与糖放入搅拌机中，打发至酥脆。 2. 加入淡奶油，搅拌均匀。 3. 过筛后加入中筋面粉，搅拌均匀。 4. 将面糊装入裱花袋中，挤入模具中（可以选择中空模具或者有模纹的模具），撒上珍珠粉。 5. 放入预热至180度的烤箱中烤18分钟左右即可。 九、披萨饼 所需材料： 面团 200克 番茄酱 适量 蘑菇 适量 奶酪 适量 鸡蛋 1个 洋葱 1/4个 青椒 1/4个 番茄 1/4个 制作步骤： 1. 面团放入碗中，加入少量盐、胡椒粉和橄榄油，揉成面团，静置备用。 2. 蘑菇切成片状，洋葱、青椒、番茄切成细条状，备用。 3. 把面团擀平，烤箱预热至220度后，将披萨饼放入烤箱350度的烤盘中，烤8分钟。 4. 烤好后，涂上适量的番茄酱，放上准备好的蘑菇、洋葱、青椒、番茄、起司和一个鸡蛋。 5. 再次放入烤箱，以220度的温度烤制8-10分钟即可。 十、芋头酥 所需材料： 糯米面 150克 椰汁 适量 糖 适量 芋头泥 150克 糯米粉 适量 油 适量 制作步骤： 1. 将糯米面、椰汁和糖混合搅拌均匀，揉成光滑的面团备用。 2. 将芋头切成块状，蒸熟后捣成泥状。 3. 将芋头泥和适量的糯米粉混合均匀。 4. 将糯米面团擀成薄片，切成小块，放入芋头泥中，包好。 5. 取出包好的芋头酥，用油炸至金黄色即可。 点心制作虽然简单，但难度并不低，需要耐心和细致，更需要用心。制作好的美食，除了美味之外，还有一份满足和幸福的感觉。希望这十款点心食谱，能够为您的生活增添一些乐趣。
