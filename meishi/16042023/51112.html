泡菜的做法大全

更新时间：2023-04-16 06:02

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

泡菜是一种韩国传统的腌制食品，口感酸爽爽的，是很多人喜欢的健康小食。下面是几种不同的泡菜做法，可以根据自己的口味选择喜欢的做法。

1. 白菜泡菜

白菜泡菜是最常见的一种泡菜，制作过程简单，味道也很好。下面是做法：

  • 材料：白菜500克、辣椒粉30克、盐30克、大葱50克、蒜末10克、姜末10克、鸡精5克、白糖适量。
  • 做法：
    1. 将白菜洗净，切成4厘米见方的小块，用盐（10克）拌匀后放置20分钟，待菜叶变软后捏一下，挤出多余的水分。
    2. 将辣椒粉、葱、姜、蒜、盐、鸡精、白糖搅拌均匀，涂在白菜块上。
    3. 将处理好的白菜放入大碗中，用手揉搓5-6分钟，直到菜叶变软，出现汁液。然后将白菜块放入干净的密封罐中，紧实压实。
    4. 放置于室温下2-3天，随时检查汁液情况，如汁液过多可倒出，反之则加盖。待味道达到理想状态后，即可放入冰箱冷藏保存。

2. 萝卜泡菜

萝卜泡菜也是一种广受欢迎的泡菜品种，萝卜的爽脆口感搭配泡菜的酸辣味道，十分开胃。下面是做法：

  • 材料：白萝卜800克、辣椒粉30克、盐30克、大葱50克、蒜末10克、姜末10克、鸡精5克、白糖适量。
  • 做法：
    1. 将白萝卜洗净去皮，切成长条状，厚度2毫米左右，放入大碗中，拌入盐和白糖，揉搓均匀。
    2. 将辣椒粉、葱、姜、蒜、鸡精搅拌均匀，涂在白萝卜上。
    3. 将处理好的白萝卜放入干净的大碗或容器内，用手揉搓5-6分钟，直到萝卜出汁。然后将萝卜放入密封罐中，紧实压实。
    4. 放置于室温下2-3天，随时检查汁液情况，如汁液过多可倒出，反之则加盖。待味道达到理想状态后，即可放入冰箱冷藏保存。

3. 榨菜泡菜

榨菜泡菜是一种非常适合做夏季小食的泡菜，口感鲜爽，味道也很好。下面是做法：

  • 材料：榨菜1斤、大葱50克、辣椒干适量、芝麻适量、盐30克、糖适量、醋适量。
  • 做法：
    1. 将榨菜用清水清洗干净后放入大碗中，加入适量的盐，搅拌均匀，静置半小时。
    2. 将辣椒干剪成小段，切碎大葱，分别放入小碗中备用。
    3. 将榨菜中的多余盐水倒掉，加入糖和醋，搅拌均匀。
    4. 将处理好的辣椒和葱拌入榨菜中，撒上适量的芝麻，搅拌均匀即可。

4. 黄瓜泡菜

黄瓜泡菜是一种口感很清爽的泡菜，非常适合夏天吃。下面是做法：

  • 材料：黄瓜2个、辣椒粉30克、盐适量、大葱适量、蒜末适量、姜末适量、鸡精适量、白糖适量。
  • 做法：
    1. 将黄瓜洗净，去皮，对半切开去籽，切成长条状。
    2. 将黄瓜放入大碗中，拌入适量的盐，揉搓均匀。
    3. 将辣椒粉、葱、姜、蒜、鸡精、白糖搅拌均匀，涂在黄瓜上。
    4. 将处理好的黄瓜放入干净的密封罐中，紧实压实。
    5. 放置于室温下2-3天，随时检查汁液情况，如汁液过多可倒出，反之则加盖。待味道达到理想状态后，即可放入冰箱冷藏保存。

5. 红薯泡菜

红薯泡菜是一种健康、美味的泡菜，口感酸甜可口，十分适合女性和儿童食用。下面是做法：

  • 材料：红薯500克、辣椒粉适量、盐适量、蒜末适量、姜末适量、鸡精适量、白糖适量。
  • 做法：
    1. 将红薯削皮，洗净，切成小块，放在大碗中，拌入盐和白糖，揉搓均匀。
    2. 将辣椒粉、蒜末、姜末、鸡精搅拌均匀，涂在红薯块上。
    3. 将处理好的红薯放入密封罐中，紧实压实。
    4. 放置于室温下2-3天，随时检查汁液情况，如汁液过多可倒出，反之则加盖。待味道达到理想状态后，即可放入冰箱冷藏保存。

以上就是几种常见的泡菜做法，希望大家喜欢。
