鲮鱼口感鲜美，最适合清炖和红烧，本文教你鲮鱼的多种做法

更新时间：2023-04-29 20:00

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

鲮鱼，是淡水鱼类中的佼佼者，口感鲜美，肉质细嫩，营养价值高，含有丰富的蛋白质、维生素和矿物质。鲮鱼最适合清炖和红烧，今天，我就来教你鲮鱼的多种做法，以期你们能在家中做一道美味的鲮鱼菜肴。

一、红烧鲮鱼

1. 准备鲮鱼、葱姜蒜、料酒、生抽、老抽、冰糖、盐、清水等食材。

2. 鲮鱼洗净，入开水锅烫至变色去腥，晾凉备用。

3. 热锅凉油，加入葱姜蒜爆香，加入料酒炒匀，加入生抽、老抽、冰糖炒匀。

4. 加入清水，放入鲮鱼、盐，烧开后转小火慢炖10分钟即可。

5. 放入碗中，放入葱姜末，淋上香油即可。

二、清炖鲮鱼

1. 准备鲮鱼、葱姜蒜、料酒、盐、清水等食材。

2. 鲮鱼洗净，入开水锅烫至变色去腥，晾凉备用。

3. 热锅凉油，加入葱姜蒜爆香，加入料酒炒匀。

4. 加入清水，放入鲮鱼、盐，烧开后转小火慢炖10分钟即可。

5. 放入碗中，放入葱姜末，淋上香油即可。

三、糟卤鲮鱼

1. 准备鲮鱼、花雕酒、生抽、老抽、糖、盐、鸡精、水、料酒、姜、蒜等调料。

2. 鲮鱼解冻后，用沸水焯一下，去掉浑浊的污物和腥味，切成适中大小的长条形，备用。

3. 捞一点糟卤（可以买来），纯净水浸泡十几分钟，去掉糟渣，加入生抽、老抽、糖、盐、鸡精、花雕酒、料酒、姜、蒜一起烧开，再加鲮鱼入锅煮开。

4. 小火慢炖七八分钟，香味浓郁，鲮鱼鲜香可口。

四、红椒鲮鱼

1. 准备鲮鱼、红椒、绿椒、葱姜蒜、盐、胡椒粉、料酒、糖、生抽、味精等食材。

2. 鲮鱼洗净，斜切成长条状，放入碗中，加入盐、味精、胡椒粉、料酒拌均匀腌制15分钟。

3. 热锅冷油，放入葱姜蒜爆香，加入红椒、绿椒翻炒。

4. 放入鲮鱼，加入料酒、生抽、盐、糖翻炒均匀即可。

五、鲮鱼丸

1. 准备鲮鱼、肉糜、料酒、盐、香菇、鸡蛋、淀粉、葱姜等食材。

2. 鲮鱼、香菇、葱姜洗净，切小粒，放入肉糜中拌匀。

3. 加入鸡蛋、盐、料酒、淀粉，加入适量冷水搅打成鲮鱼肉糜糊。

4. 热开水，用勺舀取适量的鲮鱼糜放入热水中煮熟捞出即可。

六、姜汁蒸鲮鱼

1. 准备鲮鱼、姜汁、醋、上汤（或水）、盐、葱末等食材。

2. 鲮鱼洗净备用，用刀在鱼身上轻轻划两刀。

3. 在鱼身上均匀地刷上一层姜汁，让其腌渍10分钟左右。

4. 热锅冷油，加入葱姜，倒入腌好的鲮鱼，倒入适量的水。

5. 盖上锅盖，大火烧开后转小火蒸10分钟，出锅前撒上葱花即可。

以上就是我今天分享的鲮鱼做法，每一种做法都有其独特的口感和滋味，希望大家可以根据个人口味来选择。
