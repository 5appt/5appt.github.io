湘菜做法大全

更新时间：2023-05-11 06:34

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

中国有众多的菜系，其中湘菜因为独特的口味和烹饪技巧备受青睐。下面介绍一些常见的湘菜做法，让大家可以在家也能尝到正宗的湘菜美味。

一、米豆腐炒肉丝

1. 准备食材：猪肉、米豆腐、辣椒、葱、姜、盐、酱油、料酒、糖。

2. 猪肉切丝，加盐、料酒拌匀腌制10分钟；米豆腐切块备用。

3. 热锅凉油，加入葱姜爆香，放入肉丝煸炒至变色。

4. 加入米豆腐块翻炒，加盐、酱油、糖调味，最后加入辣椒段煸炒即可。

二、毛血旺

1. 准备食材：猪血、火腿肠、白菜、豆皮、豆腐皮、红油豆瓣酱、葱姜蒜、料酒、盐、糖。

2. 猪血切块，放入滚水中焯水；火腿肠、白菜、豆皮、豆腐皮切块备用。

3. 热锅凉油，加入葱姜蒜、红油豆瓣酱炒香，加入火腿肠、白菜、豆皮、豆腐皮翻炒。

4. 加入猪血块、料酒煮沸，加盐、糖调味，煮至汤汁浓稠即可。

三、口味鸭爪

1. 准备食材：鸭爪、花椒、姜、葱、盐、料酒、老抽、生抽、糖。

2. 鸭爪冲洗干净，剁成小段，用开水焯水备用。

3. 热锅凉油，加入花椒、姜、葱爆香，放入鸭爪煸炒。

4. 加入料酒、盐、老抽、生抽、糖调味，翻炒均匀即可装盘享用。

四、辣子鸡

1. 准备食材：鸡腿肉、干辣椒、花椒、葱姜蒜、盐、糖、酱油。

2. 鸡腿肉切小块，焯水去除血水，备用。

3. 热锅凉油，加入干辣椒、花椒、葱姜蒜爆香，放入鸡块煸炒。

4. 加入盐、糖、酱油调味，最后加入适量水煮沸，关火后继续焖10分钟即可。

五、剁椒鱼头

1. 准备食材：鲫鱼头、剁椒、葱姜、蒜泥、白胡椒粉、盐、料酒。

2. 鲫鱼头洗净备用，蒜泥、剁椒、葱姜切碎备用。

3. 热锅凉油，加入葱姜蒜爆香，加入剁椒炒香后放入鱼头。

4. 加入适量水、盐、料酒煮沸，煮至汤汁浓稠，加入白胡椒粉即可。

六、蒸辣子肉

1. 准备食材：猪肉、干辣椒、花椒、料酒、盐、生抽。

2. 猪肉切薄片，加入盐、料酒、生抽腌制10分钟；干辣椒、花椒磨成粉备用。

3. 将腌制好的猪肉码在碗中，撒上干辣椒粉、花椒粉，放入蒸锅中蒸7-8分钟。

4. 出锅前淋上热油即可。

七、湘西酸肉

1. 准备食材：五花肉、腐竹、枸杞、八角、葱姜、干辣椒。

2. 五花肉切小块，煮至八成熟捞出，备用。

3. 热锅凉油，加入葱姜、干辣椒爆香，放入五花肉、腐竹、八角、枸杞炒匀。

4. 加入适量水煮沸，加盐、糖、醋调味，大火收汁即可。

八、剁椒蒸鱼

1. 准备食材：鲫鱼、剁椒、葱姜、盐、料酒、香油。

2. 鲫鱼去头、去内脏，洗净备用；剁椒、葱姜切碎备用。

3. 鱼放入盘中，加盐、料酒腌制10分钟。

4. 蒸锅加水烧沸，放入蒸鱼盘，鱼上撒上剁椒、葱姜，开小火蒸10分钟。

5. 取出蒸鱼盘，淋上香油即可。

九、红烧肘子

1. 准备食材：猪肘子、葱姜、料酒、盐、糖、酱油、老抽、香叶、八角。

2. 猪肘子洗净切块，用开水焯水备用；葱姜切碎备用。

3. 热锅凉油，加入葱姜煸炒，放入肘子爆炒至两面金黄色。

4. 加入料酒、盐、糖、酱油、老抽、香叶、八角、适量水煮沸，转小火慢慢炖煮至软烂即可。

十、麻辣香锅

1. 准备食材：牛肉、羊肉、豆皮、小白菜、粉条、腐竹、口蘑、木耳等。

2. 牛肉、羊肉切薄片，放入滚水中焯水备用；豆皮、腐竹泡发，粉条焯水。

3. 热锅凉油，加入葱姜蒜、干辣椒爆香，放入牛肉、羊肉、豆皮、腐竹、口蘑、木耳等翻炒均匀。

4. 加入盐、糖、酱油、老抽、生抽、郫县豆瓣酱、花椒粉、料酒等调味，最后加入小白菜、粉条翻炒均匀即可。

以上就是一些湘菜的常见做法，大家可以根据自己的口味和喜好进行变化，制作出更加美味的湘菜佳肴。
