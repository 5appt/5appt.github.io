CPU Identifier

更新时间：2023-05-19 01:04

应用介绍

*** 次数：2250100 已用完，请联系开发者***

使用AMD处理器的CPU Identifier的朋友应该对CBiD比较熟悉。这是一个免费的准官方测试/校准工具，专门用于AMD。Vitaliy Jungle自己写内核，自己设计界面。其主要作用是检测AMD处理器的真伪，提供相关参数和特征信息作为参考辅助！对于AMD处理器的用户来说，CBiD比cpu-z更原生，因为它获得了AMD官方的“AMD嵌入式解决方案合作伙伴”认证！而且它有一个漂亮的界面，这可能是最酷的CPU检测工具。

CPU Identifier特色

- CPU内核状况：单芯，双核，四核，八芯等;了解核心类型，更了解您的手机

- CPU负载状况：动态呈现CPU负载历史

- 浮窗观测：任何操作界面下都可观测核心的表现

-- 加快CPU速度

-- 节省电力(降低CPU频率)

CPU Identifier亮点

1、人性化搜索附近蓝牙设备，可以迅速连接上设备并进行调试

2、多种发送、接收方式，包含HEX(十六进制字节)，ASCII码等方式

3、智能记录发送的*近条目，再次发送不用重复输入

4、添加了自动分包机制，防止在特殊情况下一条消息分成两段显示

CPU Identifier优势

为用户增加了输入命令功能

在此界面，用户可以向使用linux一样操作自己的爱机，并且是以root权限进行

可能喜欢此软件的用户增加了登录功能

登录需要一杯咖啡钱，用户可以以此表达对软件的喜欢

CPU Identifier说明

1、开机自启动

2、可停用HTC设备的Perflock

3、根据系统内核和CPU架构来动检测支持的CPU频率调节值

4、支持自定义CPU频率调节

5、包含丰富的情景模式设置，用户可以根据设备和系统的条件来配置相应的CPU频率调节器，从而达到优化性能和省电的目的

6、为不同的CPU频率单独设置电压(调压)，避免因为电压过高/过低导致系统不稳定(需要系统内核支持)

7、对调节器的参数进行微调

8、查看设备/CPU/电池/状态时间/内存信息

9、执行快速/完整/原生基准测试和强度性能测试
