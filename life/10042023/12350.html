中国刷机二十年：致我们终将告别的刷机时代

日期：2023-04-10 栏目：生活常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

2023 年 1 月 7 日，魔趣 ROM 创始人马丁龙猪宣布「删库跑路」，魔趣项目停止更新。作为国内最大的安卓开源刷机项目，魔趣的结束，冥冥之中意味着，中国的刷机时代已将行落幕。

虽然一些国际上知名的刷机 ROM 在国内还备受推崇，比如 crDroid、Pixel Experience、LineageOS、Paranoid Android 等等，但往昔全民刷机的盛况，已然不复存在。

如今谈到刷机，我会想起村上春树在《1973 年的弹子球》里的一段话：

「弹子球研究专著《奖分》的序言中这样写道：除了换成数值的自尊心，从弹子球机中你几乎一无所得，而失去的却不可胜数。至少失去了时间——失去了用足以建造所有历届总统铜像的铜板都换不来的宝贵时间。」

刷机不正是这样吗？曾经的我们整日对着手机电脑，翻遍各个手机论坛，脑子里塞满了 Bootloader、Recovery、双清、root。成功进入手机桌面就欣喜若狂，循环卡开机 logo 就痛苦不堪。到最后，我们好像一无所得，却失去了难以计量的时间。

于是，我们整理了一段关于刷机的简短历史，尝试着记录刷机从兴起、繁荣到被人遗忘的前前后后。这十几二十年的故事，是一个领域的岁月史书，或许也是你我的时光故事。

从西门子 6688 到诺基亚凤凰刷机

手机刷机，和电脑装系统类似，一般指安装官方或修改版的软件系统，进行升级或修改。这个词虽然近十年才广为人知，但其概念早在 2001 年前就已经出现。那是苹果初代 iPod、微软 Windows XP 首秀登场的时代，也是许许多多 70 后 80 后玩机「老大爷」们的青春岁月。

2001 年前后，西门子先后推出 SL45、SL45i 手机，中国市场的命名相当入乡随俗——西门子 6688 和 6688i 。6688 首次支持外置存储卡和 MP3 播放，6688i 首次支持 JAVA 虚拟机，可以安装各种 JAVA 程序和游戏。在功能机和 MP3 的年代里，在手机上实现这些功能，就是智能机的体验。

不巧的是，6688 可能也是年轻人受到的第一次背刺。6688 和 6688i 配置一样，却因为发布更早，扩展内存卡容量缩水，JAVA 功能无缘。为了夺回失去的功能，数码发烧友们踏上了可能是手机史上的第一次刷机之旅。

有大神破译了 6688 系列手机的源码，并依此推出了破解的 VPK 补丁。在 6688 上刷入之后，就可以获得 JAVA 支持，安装软件游戏，可扩展外置存储卡也来到 2GB，功能大幅提升。不仅如此，它还能变身成电子书阅读器、PDA 甚至单片机编程人员的工程机。

手机一刷，就能获得缺失的功能，还有折腾的快乐。西门子 6688 开启了刷机文化，并传到了微软 Pocket PC / Windows Mobile 和摩托罗拉 MontaVista Linux 平台上。

从 2003 年开始，摩托罗拉推出了一系列 MontaVista Linux 系统手机。可能摩托罗拉的商业思路有点迂，只允许这些手机安装 JAVA 软件，而禁止了 MPKG 格式的 Linux 软件。好好的智能手机，愣是做成了功能机。

摩托罗拉A1200

于是破解玩家就出手了，在刷机 ROM 里破解集成 Linux 软件安装功能。Linux 手机的软件一下就丰富起来，比较流行的有动态桌面、主题安装、来电防火墙、屏幕截图等等。如今这些系统自带的功能，在那时还都得自己来装。昔日国民聊天软件手机 QQ 也推出过 Linux 手机版，后来弃坑转向 JAVA 版本。

Linux 时代继承神机衣钵的，是摩托罗拉 E680 系列。国内大神做的龙虾包、玩物尚志包、思云包，风靡各大论坛。

既然说到玩机，手机游戏自然少不了。因为摩托罗拉的谜之操作，想要畅玩游戏的用户就不得不去刷机，然后安装各种 Linux 游戏模拟器，比如 MAME 和 EzxVBA。MAME 模拟器支持很多的经典街机、FC、SFC、GBA游戏，给摩托罗拉 E680G 装一个，秒变游戏手机。

微软的 Windows Mobile 平台也支持刷机，相对可玩性不是那么高。老玩家们肯定还记得多普达和 HTC 经典的红绿蓝三色屏 Bootloader，后来其 Windows Phone 也沿用了这个界面。

虽然微软 Windows Mobile 的移动梦遭遇了失败，但刷机之神 HTC HD2 在这里涅槃重生。它搭载 Windows Mobile 6.5，因为无缘下一代的 Windows Phone 7 而成为末代「弃舰」。2010 年，DFT 团队为 HD2 推出了一个修改版的 Bootloader，让它成功刷上 Windows Phone。

在此后多年的时间里，各路大神轮番出手，为 HTC HD2 刷上了各种各样的操作系统，包括 Windows RT、Firefox OS、MeeGo、Ubuntu Touch，CM 和 MIUI、Flyeme 等安卓发行版，等等等等。我愿称 HTC HD2 为检验刷机神机的最高标准。

不过，当时国内玩家们最爱折腾的手机系统，绝对是如日中天的诺基亚塞班。

诺基亚手机有三种主要的刷机方式，一是国内称为凤凰刷机的 Phoenix Service Software，二是 JAF，三是诺基亚官方的 NSU 工具 Nokia Software Updater。

诺基亚凤凰刷机工具

诺基亚在刷机方面的可玩性要丰富不少。轻度一点的，换个主题、铃声，装各种字体；进阶一点的，就是卸载系统自带软件，腾点空间安装软件装游戏；再深入一点，就是安装各种破解版游戏软件。

安卓版的八门神器

随着诺基亚玩机群体的日益增长，老诺粉们的快乐老家——DOSPY 塞班论坛，在 2005 年 6 月 18 日正式创立。巅峰时期，塞班论坛有着几千万注册会员，千万级的日访问量。这里也是中国诺基亚刷机爱好者们的天国。

一套套工具，一篇篇傻瓜式的教程，多少日后的玩机高手，在这里第一次学会刷机。

CM、MIUI、魔趣、乐蛙

随着诺基亚帝国落幕，智能手机界的权杖从诺基亚塞班交到了谷歌安卓和苹果 iOS 手中。2008 年 9 月，HTC Dream 发布，在美国称为 T Mobile G1，国内用户喜欢叫它 HTC G1。它是安卓手机的起点，也是安卓刷机的开端。

发布不久之后，玩家们就发现了在 HTC G1 上获取最高权限的方法，可以对手机进行任意修改，也就是常说的 root。以此为契机，HTC G1 的各种刷机包不断涌现。早期 G1 最流行的刷机包由 JesusFreke 开发，在他停更之后，玩家们转移到了 Cyanogen（本名 Steve Kondik）开发的派生版本——CyanogenMod，简称 CM。

CM ROM 省电流畅、开机速度快、可玩性高，深得玩家喜好。加上安卓系统开源的优势，很快风靡机圈，成为刷机 ROM 的代名词。在安卓崛起的年代里，有三个「吉祥物」最广为人知。一个是谷歌官方的安卓机器人，一个是 CM 的外星人 logo「Cid」。

还有一个是谁呢？自然就是小米 MIUI 的米兔了。

2010 年 8 月 16 日，刚刚成立四个月的小米推出旗下首款产品——MIUI，基于 CM 修改定制的刷机 ROM。预感到移动互联网时代来临的雷军，早早地在安卓刷机上看到无限可能。

是如果你使用过那些年刷了 MIUI 的手机和其他手机，一分钟之内，你就会明白 MIUI 有多厉害。普通手机系统简陋，动画朴素，换个主题图标都得用第三方启动器。MIUI 直接内置花样繁多的主题，系统启动器支持多种动画，非常的炫酷，比第三方启动器玩得还花。所谓「官方逼死同人」，大抵如此。

尽管小米手机这时候还没有面世，但 MIUI 已经适配了三星、HTC、谷歌亲儿子 Nexus One 等热门机型。小米重视发烧友们的反馈，不断为 MIUI 推出新功能，每一次的更新都能带来惊喜。

一遍一遍地刷机里，大家都记住了那位戴着雷锋帽和红领巾、一起折腾安卓机的米兔。直到 2021 年的某一天，米兔因为不明原因，从 Bootloader 界面上失去了踪影。

紧随着 MIUI的脚步，国内的刷机圈开始走向繁荣。魔趣和乐蛙是早期的优秀代表。

魔趣原名 ITFUNZ 摩托罗拉论坛，由马丁龙猪创建于 2009 年 7 月，后来被大众神机摩托罗拉 Defy 给「带飞」，一路走红。2011 年，马丁龙猪成立魔趣科技，着手打造魔趣 OS，但仅过了一年便因为资金问题停更。在马丁龙猪的倔强之下，魔趣通过开源的 MoKeeOS 继续存活。

说句题外话，魔趣的投资者中有一家名为斯凯的公司。斯凯是中国山寨机时代的霸主，MTK「App Store」冒泡平台的开发商。适逢山寨机江河日下，斯凯可能也在想借机转型安卓，但最终无功而返，黯然谢幕。

和 MIUI 相似，魔趣 ROM 也是基于 CM 定制而来，只不过走的是类原生而不是深度定制道路，因而被认为是中国版的 CM。后来安卓 5.0 推出了 Material Design 质感设计，原生系统的美学被许多玩机爱好者推崇，遵循质感设计的魔趣因而收获了广泛好评。

另一边，批评魔趣的声音也不少。有人就觉得将 CM 本地化的魔趣，不过是一个 ROM 搬运工。这种观点有失偏颇。首先原生 CM 的刷机包过于精简，往往连内置输入法都是原生的，不支持中文，国内用户用起来多多少少有点麻烦。魔趣的各种本土化改进，对于降低用户操作门槛有不小的帮助。

再者说，一些热门国产手机的 CM 和魔趣 ROM 本身就是同一位开发者维护的，而且魔趣对于 CM 也有代码贡献。伴随着国产手机畅销海外，魔趣 ROM 在国外刷机爱好者间也积累了不错的口碑。

基于安卓 5.1的魔趣 OS 第三方转制版

乐蛙创始于 2011 年 4 月，三位联合创始人赵力、袁潜龙、Jonathan Li 都是手机界的大佬。相比步履维艰的魔趣，乐蛙的发迹之路要顺利得多。2012 年 7 月，乐蛙完成了来自腾讯的 5000 万元 A 轮融资，直接起飞。

乐蛙 OS 外观上没有太鲜明的风格，但集成了许多实用的功能，比如查看后台程序的「鸟瞰」，符合中国用户习惯的通讯录百家姓等等。乐蛙专注于给千元机做系统，首款适配机型是中兴 V880。

在前小米手机时代，中兴 880 系列是国产千元性价比之王，不亚于今天的红米。

比起极客们追捧的的旗舰手机，千元机其实更需要刷机——销量很高、配置稀烂、优化奇差。系统要是没经过精简优化，根本卡到没法用。就凭着对千元机的良好优化，乐蛙很快成为仅次于 MIUI 的国内第二大 ROM 提供商。

最好的时代，最坏的时代

从 2010 年到 2014 年，苹果和安卓兴起，3G、4G 更替，中国智能手机市场迎风起飞，年出货量从 3000 万部增长到超过 4 亿部，每年几乎都是倍数级增长。手机卖得火了，折腾手机的人也就多了起来。iOS 用户研究越狱，安卓用户醉心刷机，手机圈无人不折腾。

这是安卓最好的时代，也是最坏的时代。好是对刷机折腾友好，给予用户的自由度很高，也很少从硬件上限制刷机和 root。坏是坏在安卓的软硬件水平，入门和中端机配置稀烂，系统预装软件多且操作卡顿，体验非常糟糕。看看隔壁流畅爽快的 iOS，你可能就会怀疑，安卓厂商做手机的标准，是不是能开机打电话就合格？

何以解忧？唯有刷机。

像魔趣、乐蛙一样，各路民间刷机团队如雨后春笋般涌现，如点心、VIBE、新蜂，等等。也有互联网和手机大厂入场试水，比如盛大的乐众、金立的 IUNI OS。IUNI 号称要「以小米反小米」，由当时就职于金立的卢伟冰推动。因而直到今天，网友都不忘拿「反米复金」调侃他。

刷机要火，工具先行。这一时期诞生了许多影响深远的搞机工具：用来「卡刷」的第三方 Recovery，CWM 和 TWRP；root 管理工具，超级用户 SuperSU；付费软件破解工具，幸运破解器。现在回过头来看古老的 CWM Recovery，你还能熟练地双清、刷入吗？

在这些工具之上，刷机精灵等整合程度极高的一站式工具也开始出现。它们大大降低了刷机的操作门槛，让更多的小白和轻度玩家也加入到搞机群体当中。

有了工具，还得有集合各种工具以及各路大神的场所——论坛。塞班时代落幕之后，安卓网、机锋网、安智网、酷安网等以安卓为主的搞机社区开始兴起，另外还有先前提到的 MIUI 和魔趣。软件市场、资讯门户、社交论坛三位一体，是此类社区的典型结构。不知正在读这篇文章的你，是否也是曾经的水贴专家或者资深大佬？

机锋是其中比较知名的一家，巅峰时号称有 4000 万注册会员，2000 万日访问量。国内外所有的主流机型，都能在机锋论坛找到刷机包和玩机教程，或原创或搬运，我愿称之为「中国小 XDA 」。

当然，刷机的核心要素是刷机包。工具的发展不仅让刷机更简单，也让刷机包适配更方便。部分手机厂商还亲自下场，推出了 PatchROM「插桩适配」的玩法，可以基于原厂 ROM，通过反汇编方式插入第三方 ROM 修改的代码，快速适配第三方 ROM。

小米 MIUI 和魅族 Flyme 在这方面比较积极，那会很多其他品牌的手机都有适配的 MIUI 和 Flyme 正式版刷机包。

这有好处也有坏处。一方面丰富了刷机包资源，给了玩家更多的选择。你用什么手机根本不重要，只要能刷机，它就可以用上任意品牌的系统，小米 MIUI、魅族 Flyme、vivo FunTouch、OPPO ColorOS，都不在话下。

另一方面，这种便捷性也催生了技术含量低的「打包客」和「搬运工」，加入自己的植入，赚取分红。这本来不是坏事，但有的打包客毫无节制，肆意添加植入成分，严重影响了系统体验，背离了玩家刷机的初衷。

但无论如何，对于大多数个人开发者来说，开发 ROM 顶多只能当个副业，赚点外快。当手机厂商打起机海战术，民间团队光靠用爱发电来跟进适配，实在是烧不起。2014 年前后，大多数民间团队和小工作室的刷机包，基本都陷入停更，比如新蜂。

BAT 的镀金时代

智能手机是公认的移动互联网最佳入口，而刷机行业的火爆，为资本巨头开辟了抢占移动互联网市场的捷径。不用进场做手机，做刷机系统就行。作为中国互联网企业的顶流，百度、阿里巴巴、腾讯也纷纷做起刷机生意。

2011 年，阿里推出主打云端服务的阿里云 OS，不用另外安装应用就能实现很多功能。

百度则在 2012 年 6 月发布了百度云 OS，适配中华酷联米 OV 等一众安卓品牌，还花大手笔收购了点心 OS 和 91 无线。

腾讯先是在内部做了个 TITA 系统，2015 年正式推出 TencentOS 系统，简称 TOS。

系统体验方面，腾讯 TOS 可能是最好的一个，主打精简，风格清新。百度云 OS 功能齐整，高度整合百度全家桶，一度成为仅次于 CM 的国内第三方 ROM 亚军。阿里则是找上手机厂商，砸钱大搞预装系统攻势，据说一部手机补贴 30 元，后来又投了魅族 5.9 亿美元。

MIUI 的炫酷多变，Flyme 的清新流畅，CM 魔趣的纯净原生，早都深入人心了。没有特色的刷机系统，很难吸引挑剔的刷机党。

阿里云 OS 就复杂一些。刚推出时，很多人还在用 5 元 30M、10 元 70M 的流量套餐，WiFi 网络也不像今天这样随处可见，不是谁都能耗得起这么多云服务。阿里云 OS 内置服务也特别繁杂，自由度低，和搞机玩家推崇的精简、可玩性背道而驰。

预装阿里云OS 的天语 W800

更要命的是，除了魅族之外，阿里找来预装阿里云系统的厂商，都是如天语、夏新之流的 N 线手机厂。这些小厂多是冲着预装补贴来的，哪里会关心用户体验？玩家们也心领神会，买阿里云 OS 手机就图一个便宜，到手马上就刷入别的系统。

在这场淘金热中，没人能如愿以偿。

退潮

2017 年，中国智能手机出货量首次出现下跌，预示着短暂的黄金时代已经结束。那些年里，厂商用脚做手机、用脚趾丫做系统，都能赚大钱。现在为了争夺市场，留住用户，行业必须做出改变。

2011-2022 年中国智能手机单季度出货量

（单位：百万部）

尤其是在系统体验方面，手机厂商主要做了两方面的工作：对内改进第一方系统体验，对外收紧 Bootloader 权限并限制刷机。这两件事，改变了刷机行业。

Bootloader 即引导加载程序，通常简称为 BL。解锁 BL 是刷机的先决条件，无法解锁 BL 就意味着无法刷机。这是对刷机党的致命打击。

2018 年 5 月 24 日，华为发布公告称，不再对新机型提供 BL 解锁码服务，旧机型的解锁服务也会在一段时间后后终止。

大概在同一年，小米延长了申请 BL 解锁的等待时间，从 72 小时变成了 168 小时、360 小时。因为解锁 BL 会擦除手机上的所有数据，很多搞机用户为了避免倒腾数据的麻烦，新手机都得丢抽屉里吃一段时间的灰，送给半个月后的自己。

即便如此，小米对解锁总体还是开放的。小米、Redmi、一加、谷歌 Pixel，是为数不多的全部手机支持解锁 BL 且不影响保修、基本不损坏功能的品牌。

这里必须要提一嘴三星。三星手机虽然支持解锁 BL，但会触发 Knox 熔断机制，导致安全文件夹、三星 Pay 等功能失效，有时连相机也会出现故障。因为三星采用的是物理熔断机制，一旦解锁，手机失去的功能也无法失而复得。

软件供应商也没闲着。国内银行基本清一色检查手机 root 状态，检测到手机已 root 之后就无法使用。国外的软件亦无多少宽容，许多手机解锁 BL 会掉 DRM 等级，无法观看流媒体高清影片。一些国外手游也会检查手机 root 状态，更有甚者，居然还以反外挂之名，连 USB 调试都不允许打开。

那么，我们该说厂商们扼杀了刷机吗？或许是，但不完全是。

安卓刷机兴起于野蛮生长的时代，硬件配置差，软件优化糟糕，想要流畅省电基本都得刷机或者 root 折腾一番。现在手机普遍配置好了，大家自然就懒得折腾，要么用原装系统养老，要么转投 iOS 阵营。

最关键的还是厂商们在第一方操作系统的用心。以往需要刷机、root、安装框架才能实现的功能，如今原厂系统就能搞定。想要精致的美化主题和字体？我给你做。和流氓软件斗智斗勇？我来帮你。

并不是刷机被杀死了，而是我们不再需要刷机了。

CM，在 2015 和 2016 年先后获得微软和 Benchmark 的投资，试图开启商业化进程，但未能得偿所愿。2016 年，CM 服务器关闭，创始人 Steve Kondik 辞职。好在，开源社区继承了 CM 的使命，以 LineageOS 之名继续存在。

魔趣，中国版 CM，2023 年 1 月停止更新，历史存档转移到了开源仓库 SourceForge。

2015 年 3 月 11 日，百度云 OS 宣布暂停更新和维护。

2017 年 6 月 28 日，腾讯 TOS 停止服务。百度云和腾讯 TOS 存在的历史，长不过两三个安卓大版本。

乐蛙，2016 年 3 月宣布停更，重心转向海外市场，此后杳无音讯。

IUNI OS，因为母公司金立深陷内忧外患，在 2016 年黯然退场。IUNI 豪言的零添加第三方软件，多年后被魅族拾起，打造出了一代「零广告、零推送、零预装」的魅族 18。

刷机精灵，2019 年 11 月宣告停止运营。

机锋，2020 年秋关闭论坛版块，仅保留机锋门户网站。

安智，2015 年更名智友论坛，2019 年 11 月关闭，此后开设的新论坛也在 2021 年关闭。

DOSPY 论坛，随着塞班系统的没落而陷入沉寂，现转生为 DOSPY 诺基亚方舟号维持运作。

阿里云 OS 存活至今，但已经远离普通消费者，把重心转向物联网领域。

回响

发生于中国智能手机黄金时代的刷机潮流，恰如智能手机时代的嬉皮士运动。各色的刷机 ROM ，是数码嬉皮们身上的奇装异服，耳边的摇滚乐，是智能手机理想体验的化身。

而随着大潮退去，嬉皮们剪去了长发，换上了正装，奔波于手机屏幕外的生活。倒腾手机数据的繁琐，对手机变砖的担忧，让他们收起了曾经的不羁。

回到文章开头的那个问题。在多年的刷机折腾里，我们是否真的一无所得，却徒然失去了光阴？

我想，应该不是的。

2011 年 8 月，1999 元起的第一代小米手机正式发布。介绍它的功能时，小米创始人雷军说了这么一番话：

「小米手机可以随便刷机，是无锁的，你不需要破解就可以随便刷机，可以很轻松的刷成原生的安卓系统，也可以很轻松地刷成别的深度定制的系统。其实这一点在我们内部争论也很多，分歧也很大。大家说，我们为小米手机内置的服务，他一刷我们不什么都没有了吗？

我在团队内部讲，我们 MIUI 一定要做成最好的安卓定制系统。如果我们没有这样的决心，我们干脆就别做了。我相信过去的 365 天，小米的粉丝们应该理解了，MIUI 有多强的自信和实力。」

对于那时的小米，开放 Bootloader 解锁和刷机，既是对数码发烧友的致意，也是对自己的鞭策。有这样的鞭策，才会有后来的「愿你刷机半生，归来还是 MIUI」。

再回到 CM，梦开始的地方。

2009 年 9 月，CM 创始人 Steve Kondik 收到谷歌律师函，要求停止发布刷机 ROM，因为 CM 内置的谷歌专有应用并未获得授权传播。为此 CM 短暂陷入停更。

许多网友来到谷歌官方论坛，发帖声援 CM。

用户 PhoenixAG 在主题帖《不要阻止 CM》里说，有些人喜欢安卓，并不是因为官方 ROM，而是因为能使用自定义 ROM 的开源精神；只有让 CM 这样的人来发扬安卓平台的开源精神，吸引更多的人参与其中，安卓才能变得更好。

他还希望谷歌不要变成苹果，记住，勿作恶（Don’t be Evil）！

如今，谷歌的座右铭已经从「勿作恶」变成了「做正确的事」，CM 的商业化道路也以失败告终。来自 2009 年的这番话仍然挂在谷歌论坛上，时刻提醒谷歌和各大厂商，是什么让安卓成为了今日的安卓。

我们一起折腾过的时光，终归没有错付。
