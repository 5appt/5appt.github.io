日前微信安卓版推出了 8.0.35 测试版更新，官方日志称优化了一些已知问题。该版本安装包信息显示，其最低支持 API 由 21 ( Android 5.0 ) 变更为 23 ( Android 6.0 ) ，意味着 Android 5.0 及以下版本的安卓设备将无法安装。从谷歌官方博客获悉，自 Android 14 开始，谷歌将引入新的安全控制条件，禁止安装 API 23 以下版本的应用程序，推测微信正是为此而对 API 版本做出调整。
