页游也能用显卡 谷歌推出 WebGPU

更新时间：2023-04-11 06:01

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

谷歌很快将开始推出 WebGPU，该公司宣布，这项新功能将允许 Chrome 浏览器使用你的显卡来加速游戏、图形和 AI。它将在 Chrome 113 中默认启用，并将在几周内到达 Windows PC（通过 Direct3D 12）、MacOS（Metal）和 ChromeOS（Vulkan）。

谷歌表示，WebGPU 将使网络应用更好地访问你的显卡，允许开发者用更少的代码实现同样水平的图形。这可能会使基于 Chrome 浏览器的 3D 应用变得新颖有趣，毫无疑问，也会使游戏变得更好。

同时，它将为 " 机器学习模型推断的三倍以上的改进 " 提供动力，该公司补充说。这可能为本地运行的机器学习应用铺平道路，比如英伟达广播公司奇怪的 " 眼神接触 " 功能。

谷歌称最初的版本是 " 未来更新和增强的基石 "，因为开发者开始挖掘它并创建新的应用程序。该 API 已经开发了六年多，接下来应该可以在 Firefox 和 Safari 上使用（Edge 经常与 Chrome 同时获得功能），并扩展到 Android 等更多操作系统。如果你在 Chrome 测试版的轨道上，你可以使用一个名为 Babylon.js 的演示来亲自尝试这个功能，它已经提供了完整的 WebGPU 支持。
