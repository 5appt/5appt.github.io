除了优化折痕还有啥用？vivo X Fold2 铰链解析

日期：2023-04-24 栏目：生活常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

随着国内头部品牌入局折叠屏领域，折叠屏产品迎来了高速发展期，这也引得不少朋友打算下一次换机时尝试折叠屏手机。不过对于绝大多数还没使用过折叠屏手机的用户，他们最关心的莫过于折痕问题。

的确，折痕体验的好坏会影响用户对于一款折叠屏的第一印象，所以如何在寸土寸金的手机上实现良好的折痕效果，成了厂商在折叠屏产品中急需解决的问题。那么在刚刚发布的 vivo X Fold2 上，蓝厂是如何优化折痕问题的？咱们一起用实机上手看一看。

1 超耐久轻量铰链，镜面般平整

首先咱们先来看看 vivo X Fold2 的折痕表现：

vivo X Fold2（12GB/256GB）

电商报价￥8999￥8999

可以看到，我们手中的这台 X Fold2 在折痕上几乎可以做到忽略不计，不论是在观感还是触感上的体验都相当不错。

那么蓝厂是怎么做的呢？官方称 X Fold2 升级了全新的超耐久轻量铰链，才使整机带来镜面般平整的体验。

具体来说，为了更好的缓解折痕问题，vivo 将 X Fold2 的水滴铰链直径从此前的 4.6mm 升级到了 5.0mm，更大的空间让折痕更不容易产生。根据 vivo 官方数据显示，vivo X Fold2的平整度相较上一代提升约 40%。

好的折痕体验有了，我们自然要关注铰链的耐用性，毕竟大几千块钱的产品，维修一次还是挺肉疼的。为了解决这一精密的铰链结构的耐用性问题，vivo 在铰链中使用了高强度碳纤维、FS54 航空高强度钢等先进材料。不仅给 X Fold2 带来了更好的平整度、更轻的重量，还有更高的强度。

当然了，蓝厂介绍自家的折叠屏产品自然是什么好听说什么，这个超耐久轻量铰链到底怎么样我们看看专业测试机构怎么说？

根据德国莱茵实验室数据显示，vivo X Fold2 在经过 40 万次折叠实验后，仍能将折痕控制在优秀水平。这也就意味着即使按照每天折叠 100 次，十年之后 X Fold2 的折痕仍能控制在优秀水平，这对于普通用户来说完全绰绰有余。

2 平整折痕外，悬停体验不妥协

超耐久轻量铰链在带来更好的屏幕平整度、更轻的重量，更高的强度外，还给 X Fold2 带来了悬停的能力。

除了常规的悬停观影、分屏、小窗等功能外，X Fold2 新增了锁定外屏显示功能。即在锁定外屏的情况下，实现帐篷、屏风、台历等模式，为使用带来更多自由度。值得一提的是，在帐篷和台历模式下锁屏后，vivo X Fold2 还可以提供有趣的息屏时钟模式，使其成为你桌面上的好玩摆件。

另外，当我们在外面拍照时，也可以借助 X Fold2 的折叠屏形态，将手机折叠到 90 度进入悬停自拍模式，搭配手势快门功能，轻松拍出好看自拍。当然我们也可以使用后置镜头，利用悬停模式取代三脚架拍摄夜景或是长曝光照片。

写在最后

根据 vivo 官方介绍，之所以 X Fold2 可以实现如此之多的悬停体验，除了超耐久轻量铰链本身外，vivo 在 X Flod 的阻力结构上尝试过 5 种材料（弹簧三种，凸轮两种），5 种不同的润滑脂，进行了 20 多版设计，做了 7 次实物样品、50 次模拟仿真，8 次开模，才最终实现了现在的完美效果。
