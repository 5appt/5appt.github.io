仅 1 厘米薄 1 千克轻，华硕 Zenbook S 13 OLED 笔记本发布

日期：2023-04-24 栏目：生活常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

4 月 23 日消息，华硕 Zenbook S 13 OLED 发布，这款笔记本标重为 1.00 千克，尺寸（宽 x 深 x 高）为

29.62 x 21.63 x 1.09 ~ 1.18 厘米。在目前上市的笔记本中，华硕 Zenbook S 13 OLED 在厚度和重量上都做到了 TOP 级。

目前仅在华硕在线商店订购，价格为 1399.99 美元（约合 9650.97 元人民币）。

Zenbook S 13 OLED ( UX5304 ) 采用 13.3 英寸 2.8K ( 2,880 x 1,880 ) Lumina OLED 显示屏，16:10 宽高比，0.2ms 响应时间，60Hz 刷新率。显示屏通过杜比视界 ( Dolby Vision ) 和 VESA DisplayHDR True Black 500 认证。它拥有 550 尼特的峰值亮度，覆盖 100% 的 DCI-P3 广色域，屏占比为 85%。

处理器有两个版本：

英特尔 酷睿 i7-1355U 处理器 1.7 GHz（12MB 缓存，高达 5.0 GHz，10 核，12 线程）

英特尔 酷睿 i5-1335U 处理器 1.3 GHz（12MB 缓存，高达 4.6 GHz，10 核，12 线程）

最高可配置 32GB 板载 LPDD5 内存和高达 1TB PCIe Gen4 固态硬盘。其它配置包括英特尔 Iris Xe 显卡、Wi-Fi 6E、蓝牙 5.2 以及 63 Wh 电池。

接口方面，两个 Thunderbolt 4 Type C、一个 USB 3.2 Gen 2 Type A、一个 HDMI 2.1 ( TMDS ) 和一个 3.5mm 耳机口。
