「有料评测」vivo X Flip 全面评测：不做美丽小废物

更新时间：2023-04-22 07:14

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

去年 4 月，vivo 带来了自家首款折叠屏手机 X Fold，几乎找不到短板的硬件配置、系统层级折叠屏软件优化，给我留下了深刻印象。而就在刚刚结束的 vivo 折叠系列旗舰新品发布会上，蓝厂不仅更新了新款的 X Fold2，还带来了首款竖向折叠形态产品 vivo X Flip。

那么作为头部品牌中的第四款竖向折叠屏，vivo X Flip 的具体体验如何？又有着哪些体验上的优势？值不值得购买呢？今天我来带大家一起上手体验一下。

01 不仅是手机，更是 " 轻奢小包 "

首先咱们先来看看 vivo X Flip 的外观。

vivo X Flip 的机身采用了竖向折叠结构设计，198g 的机身重量和圆润的机身曲线，勾勒出了优秀的握持体验。尤其是折叠起来后，75mm*86mm 的小方块机身像是一个粉饼盒，让人忍不住想多把玩一会儿。

配色设计上，vivo X Flip 带来了菱紫、绸金、钻黑三款配色，其中菱紫配色采用了素皮材质，并且在素皮表面融入小皮包中常用的菱格纹元素，颇有几分奢侈品小包的感觉。 另外针对于素皮材质的特性，vivo 在素皮表面加入了耐磨抗脏污纳米涂层，进一步提升了其的耐用性。

绸金和钻黑配色的 X Flip，分别采用了空间绸光纹理工艺和晶钻切面纹理工艺，简单来说就是亮面的玻璃材质，整体的质感相当不错，同时在触感上更加丝滑。

聊完了手感，咱们再来仔细看看 vivo X Flip 的机身，这里我们以菱紫为例。

X Flip 机身背部的上半部分是一块 3 英寸外屏、影像模组以及柔光灯，下半部分则完全被菱格纹素皮包裹，"vivo" 的品牌 logo 置于中心靠下的位置。

机身右侧是音量加减键和集成了指纹识别的电源键；机身底部是 SIM 卡卡槽、Type-C 接口以及扬声器。

X Flip 的机身正面是一块居中挖孔的 6.74 英寸超视网膜护眼屏，支持 120Hz 刷新率、1920Hz 高频 PWM 调光，同时这块屏幕采用了蓝钻排列，观感上还是相当细腻的。

编辑

总的来说，vivo X Flip 的机身秉持了 vivo 一贯精致的风格，并且创新性的带来了 " 奢牌小包 " 的设计理念，只能说蓝厂太懂女孩子了。并且值得一提的是，对于没有选择菱紫配色的用户，vivo 同样贴心地带来了小荔枝纹素皮材质的 " 轻奢小包 " 手机壳和珍珠星环链，秒变 " 奢牌小包 " 不仅是素皮版本的特权。

02 不仅是小折叠，更是好用的小折叠

作为一款竖折产品，很多朋友应该会比较在乎折痕的问题。不过从去年发布的 X Fold 系列来看，蓝厂对于折痕的优化还是有一套的。

根据官方介绍，这次的 vivo X Flip 搭载了 5.3mm 超大直径水滴铰链，相较于 " 小水滴 "，X Flip 在折痕上优化了 44%。

不看官方的套话咱们直接在光线下给大家看看 X Flip 的折痕表现，基本是可以忽略的，结合我最近的使用体验和评测过的其他折叠屏产品来看，已经算的上很不错的水平了。

以绝大多数用户最常用的微信来说，这块外屏不仅支持微信聊天，还可以翻看历史聊天记录、点开图片、抢红包、刷朋友圈等操作，甚至你还可以和别人用表情包斗图。基本我们日常使用微信的绝大多数场景都能实现，唯一遗憾的是尚不支持微信视频通话，后续可能才会进行官方适配。

另外，X Flip 比较人性化的一点是，这块外屏单手操作时无需区分左右手，不论是横向使用还是竖向使用，都有良好的交互体验。

值得一提的是，官方称单独使用这块外屏时，当手机电量剩余 10% 时，外屏还可以有 10.28 小时待机时间。这也就意味着即使电量告急，合起 X Flip 使用外屏依旧可以有较长的续航时间，以应对一些燃眉之急。

03 要美拍，更要优雅

除了设计，影像表现往往是 X Flip 这类竖折产品受众第二关心的卖点。

咱们先来看看硬件部分，vivo X Flip 的后置影像模组由一颗 5000 万像素的仿生光谱大底主摄和 1200 万像素的超广角镜头组成，前置则是一颗 3200 万像素的高素质镜头。

不过对于竖折形态的产品，后置镜头不论是自拍还是日常拍摄，使用的频率会更高一些，所以这里我们着重介绍后置影像模组。

05 总结：量大管饱的小玩意儿

作为蓝厂的首款竖折形态的折叠屏，X Flip 的综合体验还是相当棒的。轻奢小包的设计理念和超耐久轻量铰链的加持，搭配上蔡司影像系统、骁龙 8+ 旗舰芯片、4400mAh+44W 的续航组合，即使之前一直在使用直板手机，选择 vivo X Flip 当做主力机来使用也是一点儿问题也没有的。
