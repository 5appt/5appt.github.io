大疆Mavic 3 Pro的三摄系统

大疆Mavic 3 Pro是首款消费级无人机中拥有三摄系统的产品，开创了消费级无人机的三摄时代。从相机部分，我们可以直观的辨认这款无人机是否为大疆Mavic 3 Pro。

大疆Mavic 3 Pro的三颗摄像头包括4/3画幅CMOS传感器的主摄，采用了哈苏相机、2000万像素、等效24mm焦距、f/2.8-f/11可变光圈设计，能够拍摄最高5.1K 50P和4K 120P规格视频的录制，原生12.8档动态范围，支持哈苏自然色彩解决方案（HNCS），支持D-Log色彩模式和新增加的D-Log M色彩模式。

另外两颗摄像头为中长焦相机以及长焦相机。中长焦相机采用了1/1.3英寸CMOS传感器设计，等效70mm焦距、4800万像素、3倍光学变焦、f/2.8光圈设计，能够记录最高4K 60P视频规格，并支持新增加的D-Log M色彩模式。

长焦相机在Mavic 3中就已经出现，在大疆Mavic 3 Pro得到了升级，目前的配置是1/2英寸CMOS传感器设计，等效166mm焦距、1200万像素、7倍光学变焦、28倍混合变焦、f/3.4光圈设计，支持最高4K 60P视频规格。

三个摄像头之间的切换也很简单，只需要通过遥控器界面上的1x、3x和7x图标点击即可。

大疆Mavic 3 Pro的桨叶设计

大疆Mavic 3 Pro的桨叶设计也与Mavic 3家族保持一致，同样是安全桨叶。所谓安全桨叶，也就是桨叶的末端材质与桨叶本身材质不同，而是塑胶材质，也就是图中橙色的部分。这个橙色部分是软的，因此飞行过程中即使划过物体，也只会留下痕迹，不会直接割破物体。也就是说在飞行过程中，如果真的与人员和动物、植物接触发生意外，也不至于造成过大的伤害。但是笔者建议大家飞行时要谨慎，避免造成伤害。

大疆Mavic 3 Pro的接口和卡槽

大疆Mavic 3 Pro的卡槽和接口设计在电池仓的上方，并且设计了保护罩，这样可以做到防尘、防水溅。接口是USB Type-C接口，可以直接充电或者连接电脑导数据，卡槽是TF（Micro SD）卡槽。另外，大Mavic 3 Pro本身拥有8GB内存容量，可以应急使用。

我们重点来看看大疆Mavic 3 Pro的两个长焦相机的表现。

我们先说说大疆Mavic 3 Pro的7倍光学变焦、28倍混合变焦的长焦相机，这款相机的传感器与Mavic 3是同款，但是它的等效焦段有了延长，从162mm延长到166mm；同时光圈明显改善了，从f/4.4增加到f/3.4。这个改变的优势就是，这个长焦相机的画质得到明显的改善，当然光线强烈的环境下差距并不会很明显，但是在光线昏暗的环境下，比如夜间，它的画质水平的确有了进步。对于广大用户来说，可以更好的去利用这个长焦相机来完成远距离取景拍摄的需求。

看了这个视频，相信很多人都会理解大疆将Mavic 3 Pro升级到三摄系统的意义了。三摄系统的三个相机，分别拥有差异比较大的等效焦距设计，分别为24mm、70mm和166mm，在空中航拍的时候，用户可以记录这三个等效焦段的画面。24mm记录宏伟的大场景、70mm记录主体与环境的位置关系、166mm则可以完成局部特写的记录。

对于用户实际操作来说，在大疆Mavic 3 Pro位置不变的情况下，就可以通过切换不同的相机来记录不同构图的画面，在很多环境下，这样的操作可以让无人机更安全，同时也更节省时间和电量。当我们拍摄一个相同大小的主体时，由于使用的镜头的等效焦段不同，也可以记录主体与环境的不同位置关系，比如使用长焦相机可以让主体与环境之间的位置变化更紧凑，画面感更震撼，而广角则是交代主体在大环境中的所处位置。

综合来看，这次大疆Mavic 3 Pro最大的差异就是影像部分，通过我们的实际体验，的确对得起它旗舰级的定位以及目前的配置表现。除了给用户带来了更多画面焦段，也就是创作视角的选择之外，这一次整体的画质表现都非常出色，相信每一个相机的利用率都会很高，对于用户创作来说具有非常明显的优势。使用大疆Mavic 3 Pro创作的专业航拍视频，可以获得更好的表现以及更丰富的画面。

大疆Mavic 3 Pro旗舰无人机

因为大疆Mavic 3 Pro的到来，正式开启了消费级无人机的三摄时代。三摄系统的意义是什么，其实就是进一步提升创作者的拍摄效率，可以让用户在距离不变的情况下，直接体验不同焦段视角所带来的不同画面感。主摄广角相机可以记录更丰富的景色，交代整体大环境；中长焦相机可以交代拍摄主体与周围环境的位置关系；长焦相机则可以完成局部特写。在这个过程中，无需更换镜头，通过点按大疆Mavic 3 Pro遥控器屏幕便可切换不同焦段，画面流畅变换，满足不同场景下画面的自由表达，让创作本身有更多的不同风格。
