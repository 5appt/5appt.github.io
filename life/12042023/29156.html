实测对比：游戏本最优之选为何是 RTX 4050 和 RTX 4080？

日期：2023-04-12 栏目：生活常识 浏览：次

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

RTX 40 游戏本的首批产品基本已经全面上市，从入门级的 RTX 4050 到高端的 RTX 4080，新一代 GPU 可以说是给游戏本带来了飞跃性的图形性能表现。优势在支持 DLSS3 的情况下，RTX 40 游戏本在光追游戏方面可以说是大幅提升，即便是入门级的 RTX 4050，都能借助 DLSS3 获得非常出色的游戏体验。

那么目前主流的 RTX 4050、RTX 4060、RTX 4070 以及 RTX 4080 四款独显，其性能到底怎样呢？与前代产品相比提升有多大呢？今天我们就针对全新的 RTX 40 系列笔记本电脑 GPU 的性能，做一个系统性的测试。

首先我们还是先来看看四款不同型号独立显卡的基本规格参数，具体如下：

首先是 GeForce RTX 4050 笔记本电脑 GPU，它拥有 2560 个 CUDA 核心，6GB GDDR6 显存，像素填充率 113 GPixel/s，纹理填充率 188.4 GTexel/s，总线位宽 96bit，核心频率 2070MHz，显存频率 2000MHz，Boost 频率 2355MHz。

RTX 4060 笔记本电脑 GPU 拥有 3072 个流处理器，8GB GDDR6 显存，显存位宽 128bit，基础频率 2175MHz，加速频率 2370MHz，这是架构工艺改进带来的游戏性能的明显进化。

GeForce RTX 4070 笔记本电脑 GPU 拥有 4608 个 CUDA 核心，核心频率 2170MHz，Boost 频率 2275MHz；显存为 8GB GDDR6，来自三星，显存位宽 128bit，显存频率 2125MHz。

RTX 4080 笔记本电脑 GPU 满血功率为 175W，拥有 7424 个 CUDA 核心，12GB GDDR6 显存，像素填充率 154.4 GPixel/s，纹理填充率 447.8 GTexel/s，总线位宽 192bit，核心频率 1510MHz，显存频率 2250MHz，Boost 频率 1930MHz。

全新的 RTX 40 系列笔记本电脑 GPU 之所以性能大幅提升，与英伟达最新的 Ada Lovelace 架构密不可分，它不仅拥有新型 SM 多单元流处理器，同时 Tensor Core 和 RT Core 也得到了升级换代。此外，前面提到的 DLSS3 技术对光追游戏体验提升十分显著。DLSS3 包括帧生成技术、超分辨率技术、以及 NVIDIA Reflex 低延迟技术，可以在保证优秀图像质量的同时，大幅提升性能表现，使得全景光线追踪游戏成为现实。其中帧生成技术也就是大家常说的插帧，它通过 AI 实现了人眼难以分辨的效果。再加上 NVIDIA Reflex 低延迟技术，可以将因插帧带来的延迟降到最低。在这些技术的加持下，DLSS3 可在部分游戏场景下实现数倍提升。

接下来我们看看 3DMark 各项跑分测试结果对比，具体如下：

从 TimeSpy、PortRoyal 以及 FSE 三项测试来看，RTX 4080 笔记本电脑 GPU 的性能可谓是一骑绝尘，相比桌面级 RTX 4080 的坑爹性价比，笔记本端的 RTX 4080 是真心不错，整体性能较上一代旗舰 RTX 3080Ti 而言提升约 40%，性能表现优异。

此外，因为 RTX 40 系显卡全面支持 DLSS3，因此即便是 RTX 4050，在面对光追游戏时也是游刃有余。我们以《赛博朋克 2077》为例，看看 RTX 4050+DLSS3 的效果究竟如何。

1080P 分辨率 + 超级光追模式，DLSS 自动的情况下，平均帧数可以达到 49.41fps，与 RTX 3060 独显运行这款游戏的帧数差不多。

之后我们在游戏设置里将 DLSS Frame Generation 选项打开，就会开启 DLSS3，如果不打开就是默认的 DLSS2。

相同设置开启 DLSS3 之后效果可以说是非常显著，平均帧数一下就提升到了 73.91fps，提升幅度约为 50%，这效果只能说是太强了。

· 结语

通过测试可以看到，如果单纯是玩游戏的话，其实选择 RTX 4050 机型就足够了，但是如果预算充足，想要追求更加极致的图形性能，那么 RTX 4080 机型绝对是首选，毕竟 RTX 4060 和 RTX 4070 与上一代旗舰的性能还有重合的部分，而 RTX 4080 则是全面大幅超越上一代的型号。
