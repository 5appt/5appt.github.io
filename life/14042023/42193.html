8K 全画幅空中电影机 在横店体验了大疆 Inspire 3 无人机新品

更新时间：2023-04-14 16:01

大家好，今天小编给大家整理了下面相关知识点，如果碰巧解决您的问题，别忘了关注本站哦，现在一起来看看吧

熟悉电影行业和专业视频领域拍摄的朋友，一定对大疆 Inspire（大疆悟系列）印象深刻。现在，历经 6 年多时间，大疆 Inspire 3（大疆悟 3）终于正式发布。其实在发布之前，大疆 Inspire 3 就已经参与了很多知名影视作品的拍摄，深受业内人士的肯定。在 3 月底，笔者受到大疆邀请，来到横店简单体验了大疆 Inspire 3，这里与大家一起分享一下非影视行业人士的感受。

不是吹，真的是空中电影机

2014 年大疆 Inspire 1 诞生，自此开启了 4K 影视航拍时代，后续推出 M4/3 相机，无损 4K RAW 视频录制，满足专业影视的航怕需求。2016 年，大疆 Inspire 2 发布，搭配大疆禅思 X7 相机，可实现最高 6K CinemaDNG / RAW 和 5.2K Apple ProRes 视频，飞行性能和其它方面也有质的飞跃。现在，2023 年，大疆 Inspire 3 再次实现质的飞跃，提供了最高 8K/75fps ProRes RAW、8K/25fps Cinema DNG，4K/120fps ProRes RAW 视频规格的拍摄能力，成为了 8K 专业影视航拍中，目前最佳的解决方案。

接着，我们先来看看大疆 Inspire 3 本身的硬件和能力表现。

先从大家最感兴趣的影像系统开始。大疆 Inspire 3 搭载的是 X9-8K Air 全画幅 CMOS 云台相机，目前采用了 DL 卡口系统，因此可以连接 DL 卡口镜头（目前有 DL 24mm/35mm/50mm F2.8 全画幅镜头群），而且伴随而来的还有大疆 DL 18mm F2.8 全画幅镜头，后续还会新的推出长焦镜头。

在拍摄性能方面，大疆 Inspire 3 具备双原生 ISO（在全画幅 30fps 及以下时，具备 800 和 4000 两档原生 ISO），在昏暗环境拍摄可以实现更低的噪点，保证画面的纯净度，这在专业影视拍摄方面十分重要。Inspire 3 具备 14+ 档动态范围，可以很轻松的在大光比环境下拍摄，比如日出、日落等场景。与专业的电影级规格类似，搭配 DJI 电影色彩系统（DCCS），Inspire 3 所记录的专业视频在后期和调色方面更加得心应手。

接着是 Inspire 3 本身的飞行控制镜头部分，也就是 FPV 镜头。FPV 镜头与 X9-8K Air 云台相机不同，云台相机是为了拍摄专业的影视画面而设计，FPV 镜头就是给操控 Inspire 3 的专业飞手所准备的，让飞手可以实时的观看 Inspire 3 的飞行状况。FPV 镜头具备 161 ° 超广角视角，1/1.8 英寸传感器，实现全高清 60fps 低延时图传。FPV 镜头的夜间观看效果也非常出色，因此应用在拍夜戏和暗光环境飞行上也很轻松。

大疆 Mavic 3 系列都是全向感知避障系统了，Inspire 3 自然也是如此。大疆 Inspire 3 采用了 6 个鱼眼镜头、2 个广角镜头、一对 ToF 传感器的避障系统设计，能够实现全向视觉感知避障。与我们常见消费级无人机不同的是，大疆 Inspire 3 是全球首个将视觉感知系统放置在变形机构上无人机。也就是说大疆 Inspire 3 的有 4 个鱼眼镜头分别设计在螺旋桨下方的起落架上，另外 2 个在腹部，2 个广角镜头在顶部。这样的设计可以保证 Inspire 3 的避障系统拥有更好的视角，实现更全面的感知能力。

遥控器方面，搭配大疆 Inspire 3 而来的是 DJI RC Plus 遥控器。这款遥控器的体积明显比我们常见的消费级无人机的遥控器要大很多，上面的按键设计更多，毕竟需要操控的功能也更丰富，而且自定义功能按键的数量也明显更多，可以满足影视拍摄时的很多自定义设置。RC Plus 遥控器内置 7 英寸高亮屏，显示效果非常好，在户外强光下也可以看的很清晰。遥控器本身设计了内置和外置电池，因此没电的时候也可以快速更换电池，避免耽误拍摄进度。

搭配遥控器的是 DJI O3 Pro 图传系统，这套图传系统可以实现最远 15 公里的图传距离，实现双控独立链路，双控 4G 图传。什么是双控独立链路，双控 4G 图传，这里卖个关子，我们在下文谈体验感受的部分跟大家细说。

大疆 Inspire 3 具备双 RTK 叠层陶瓷天线设计（Inspire 3 是首款采用双 RTK 天线设计的航拍无人机），能够实现厘米级的定位，并且还可以连接 4G 图传系统，即使在横店这样干扰严重的影视基地，也没有断过图传。为什么这么设计呢，因为大疆 Inspire 3 新增加了航点飞行 Pro 技术，可按照预设的飞行高度、速度、云台角度和相机设置在规划的航线上自动飞行，也就是轨迹复演模式和 3D 轨道模式。Inspire 3 的 3D 轨道模式可以模拟伸缩炮、索道和轨道的拍摄效果，这些功能也是专业影视团队呼声非常高的一个应用功能。

电池方面，大疆 Inspire 3 具备双电池供电设计，支持电池热插拔，和大疆行业无人机有异曲同工之妙。在影视拍摄时，拍摄也是争分夺秒的进行，Inspire 3 支持电池热插拔，没电的时候直接降落，不用关机，直接更换电池，这样然后直接起飞，一次更换电池至少节省了几分钟的时间。由于是双电池设计，我们在换电池时拔下一块电池，另一块电池依然在供电，这样直接更换新的即可，另一块电池也是如此，这就是热插拔的原理。另外，Inspire 3 的续航时间延长至 28 分钟，应对一组长镜头的拍摄也很轻松。

在横店体验大疆 Inspire 3 的感受

由于体验的时间非常有限，所以笔者自己只有大约 20 分钟的体验时间。整体上主要体验了一下操控 Inspire 3 飞行以及航拍这两个方面。

目前的专业影视航拍中，除了使用大疆 Inspire 2 之外，很多场景需要使用大型无人机挂载电影机来进行航拍，但是这种解决方案需要耗费大量的人力、物力和时间，而且拍摄也有一定的限制。现在，大疆 Inspire 3 或许会成为专业影视航拍的最佳一体化解决方案，同时 Inspire 3 和云台相机的体积控制还是十分到位的，因此低空拍摄也不会造成扬尘等问题出现。

相比 Inspire 2，大疆 Inspire 3 的一体化流线型设计让它具有很高的辨识度，也正是因为全新的空气动力学，让飞行气阻进一步降低， 因此 Inspire 3 的斜下降速度提升到 10 米 / 秒，垂直升降速度提升到 8 米 / 秒，机动性非常好，可以快速反应来完成特殊场景的拍摄。

在笔者实际操控 Inspire 3 飞行的体验时，虽然与消费级无人机有差异，但是整体操控的感受是一样的，逻辑也是一样的，因此如果有大疆无人机的操控经验，可以更好的上手。Inspire 3 本身的机动性很好，切换到运动模式之后，可以明显感受到无人机更加灵活，对于快速跟拍等镜头的记录来说很轻松。在精细操作飞行的时候，Inspire 3 的图传系统响应非常快，的确做到了低延时，因此近距离拍摄、狭小空间拍摄等场景时也可以实现实时的精细操作，不用担心延时造成危险的问题。

避障方面，大疆 Inspire 3 可以通过遥控器来更改实际的避障范围，对于笔者来说是全新的体验。同时，飞手可以通过遥控器左下角的实时障碍物显示图中，清晰的看到周围大致环境和障碍物与 Inspire 3 的位置关系。

另外，DJI RC Plus 遥控器虽然已经拥有轻量化的设计，但是毕竟体积硕大，重量肯定比普通消费级无人机的遥控器要重一些，不过它拥有肩带和腹托设计。长时间使用的时候，有了肩带和腹托的帮衬，我们基本上不用握持遥控器，只需要操控上面的按键和拨杆即可。遥控器配备了 HDMI 接口，可以实时的将画面输出到监视器，便于导演查看画面。

关于遥控器，Inspire 3 也为笔者带来了另一个全新的体验，就是 Inspire 3 可以同时连接两台 DJI RC Plus 遥控器。两台遥控器可以分别独立操控无人机、独立控制云台相机，也就是 Inspire 3 的飞手和云台手是两个独立个体，并不需要处在同一位置控制，并且飞手和云台手还可以相互切换。这种设计还有一个好处，如果飞手的遥控器突然断连，那么云台手可以临时接管飞行控制权，避免出现意外。

在横店现场，大疆的工作人员还演示了轨迹复演模式和 3D 轨道模式。轨迹复演模式就是让 Inspire 3 按照预设的飞行高度、速度、云台角度和相机设置在规划的航线上自动飞行。这种飞行模式可以帮助摄影师在同一位置完成完全相同的飞行和拍摄，对于拍摄日夜交替、四季交替题材的视频来说非常实用。

对于影视航拍的拍摄来说，3D 轨道模式非常实用，笔者在现场除了操控 Inspire 3 自行飞行之外，还体验了这个 3D 轨道模式。3D 轨道模式，是在现场工作人员完成航线设定之后，笔者自己来控制 Inspire 3 在同一航线上重复飞行。与轨迹复演模式不同的是，这个飞行可以有飞手控制，主要是控制 Inspire 3 的飞行速度，以及前进或者倒退飞行。这个模式主要是为了同一镜头多次拍摄准备的，也就是我们常说的第一场第一镜 Take 1、Take 2 等等。在复拍的时候，我们可以根据演员这一次拍摄的不同状态、速度来动态的改变 Inspire 3 的飞行速度，同时云台手也可以自由的控制云台来进行不同风格的创作。

在操控云台拍摄方面，笔者进行了简单的体验。Inspire 3 可以连接遥控器操控云台，也可以连接大师摇轮操控云台。现场有专业的工作人员为我们演示了通过 Inspire 3 搭配大师摇轮进行一段专业影视片段的拍摄，这套组合搭配不论是模仿地面轨道跟拍，还是空中俯拍，环绕都非常轻松，可以在一定程度上满足从地面到空中双重拍摄需求。笔者自己体验了遥控器操控云台，操作逻辑很简单，上手很快，但是如果要达到专业水准还需要苦练，总体的操控体验很好，笔者即使经验不足也可以自己实现一些简单的运镜拍摄。

总体来说，Inspire 3 是笔者目前体验的最完美的大疆无人机之一，不论是无人机本身还是影像系统的输出表现，对于影视行业、专业视频领域来说，Inspire 3 是一款革命性的产品，接下来 Inspire 3 的身影肯定会经常出现在专业录制的场景中。

影视行业内摄影指导怎么看 Inspire 3

在横店的体验现场，大疆还邀请了国内知名的几位摄影指导，包括赵晓时、赵小丁和刘寅。赵晓时是国内数一数二的摄影指导，他的代表作包括《梅兰芳》、《建党伟业》、《夺冠》；赵小丁摄影指导的代表作包括《满江红》、《影》、《悬崖之上》；刘寅摄影指导的代表作包括《流浪地球 1》、《流浪地球 2》、《外太空的莫扎特》。通过摄影指导的问答环节，我们也了解了在专业影视拍摄范畴内，Inspire 3 所带来的优势和改变。

知名摄影指导赵小丁（图片由大疆官方提供）

担任《满江红》摄影指导、是张艺谋导演多年搭档的赵小丁提到，"Inspire 3 的出现，完全可以匹配专业的大型摄影机。比如说精准地设计一条飞行的轨迹替代我们的索道、伸缩炮，像这次《满江红》开篇的第一个镜头，大院子里整个的运动轨迹非常规整和非常严格严谨，而 Inspire 3 的那种感觉真是像索道一样，稳定性非常不错。"《夺冠》的摄影指导赵晓时提到，"Inspire 3 可以重复轨迹的功能，我觉得已经不仅仅是创作自由，更像是往完美方向更近了一步。"

《流浪地球 2》的摄影指导刘寅提到，"（有了）Inspire 3 的这种航点功能，我们可以设计一条很复杂的从地面到高空的长镜头调度，可能是在街道中完成 360 ° 的环绕，包括大距离的位移的镜头运动，再加上从地面到高空的连续长镜头复杂调度，我觉得用 Inspire 3，至少基于它现在的安全性、轨迹的可提前预知性，能很好地实现这种镜头，以往的航拍会出于各种原因舍弃掉，但有了 Inspire 3 以后，这方面我可以有一些新的镜头的运动语言去突破。"

《满江红》的摄影指导赵小丁提到，"Inspire 3 比较完美地能把现场的明暗关系和色彩的呈现比较忠实地记录在我们的原始素材的存储卡上。"《流浪地球 2》摄影指导刘寅，" 让我比较惊喜的是 Inspire 3 对暗部的的处理比 Inspire 2 要进步了很多，从黑到中灰的过渡也好了很多，当然它的色彩还原比之前也更准确了一些。" 刘寅提到，"Inspire 3 对光影的控制也有提升。"《满江红》摄影指导赵小丁表示，" 大师摇轮和 Inspire 3 的匹配，相对来说就能更加精准地使我们的画面、构图都更加完美。"

赵小丁表示，"Inspire 3 的这种小型、轻便、灵活，电池、供电都能满足我们的需求，地面、空中可以自由的连接和转换。有个概念叫无障碍运动，悟系列的飞行系统在一定程度上也朝着这个目标在发展，我们将来在现场真是你想怎么运动，就能随心所欲地完成摄影师、摄影指导的初衷。"

知名摄影指导刘寅（图片由大疆官方提供）

刘寅提到，" 以往你需要特别大幅度的高低运动或者横向运动，以往是解决不了的，现在我们就可以通过 Inspire 3 来去实现这种大幅度运动的延时拍摄。Inspire 3 整个的一体化的设计可以大大减少我们现场的准备时间，几乎是拿出来、展开、通上电就可以进行拍摄，这个是对整个工作效率的提升。"

赵小丁，" 大疆是一个做增量的企业，不是做存量的，大疆不断地能创造一些价值，我觉得这个是比较重要的。电影这门艺术从百多年前诞生的那天起，它的发展到今天的核心动力就是工业和技术革命。无人机的系统同样在一定程度上扮演了这个角色。大疆的产品能有一些颠覆性的影响，为我们的电影行业有一些贡献。"

刘寅，" 我觉得特别引以自豪的是我们自主国产企业去真正地从行业的发展为目标，去做一些产品的研发，投入到我们一线的电影制作里面，这个是我觉得特别自豪的一件事。大疆还是给我们中国电影行业的国产化提供了很多的信心和未来可以看到的各种无限的可能。"

知名摄影指导赵晓时（图片由大疆官方提供）

赵晓时，" 一开始大家都是国外的东西，这些年我觉得国内的设备确确实实有长足的进步，所以我们也可以特别自豪地有我们自己的一些工业产品在我们自己的创作过程中去发挥作用。"

最后，笔者很期待未来更多的电影使用大疆 Inspire 3 来完成更多镜头的拍摄。
